# 十二大従星（異形配下）

十二支から生まれた「エネルギーの具現」。
**十二司星官に封じられており、司星官を倒すことで解放される。**

十大主星と異なり、完全な人型ではなく「異形」。
しかし人格と意思を持ち、刻夜に忠誠を誓う。

**陽支 → 男性的 / 陰支 → 女性的**

★印の三体（シズメ、オサメ、サスラ）は冥界に近い存在。
封印されていないが、**第三部以降に登場**（序盤は十大主星に集中）。

---

## 存在理由

### なぜ十二大従星は存在するのか

十大主星が「精神の具現」であるのに対し、十二大従星は「時の流れ（エネルギー）の具現」。

| 存在 | 本質 | 役割 |
|------|------|------|
| **十大主星** | 十干から生まれた「精神」 | 空白の王の「意志」を実現する |
| **十二大従星** | 十二支から生まれた「時・循環」 | 空白の王の「力」を供給する |

### 人の一生を表す存在

十二大従星は「胎→嬰→児→少→青→壮→帝→老→病→死→墓→絶」という人生の循環そのもの。

```
胎（ミナモト）：まだ生まれていない可能性
  ↓
嬰（カガミ）：生まれたばかりの白紙
  ↓
児（ホタル）：世界を知りたい好奇心
  ↓
少（カスミ）：夢と現実の境界
  ↓
青（ホムラ）：燃え上がる青春
  ↓
壮（イワオ）：守るべきものを持つ壮年
  ↓
帝（スメラ）：人生の頂点
  ↓
老（オキナ）：経験を伝える老年
  ↓
病（キザシ）：衰えゆく肉体
  ↓
死（シズメ）：終わり
  ↓
墓（オサメ）：眠り
  ↓
絶（サスラ）：そして再び始まりへ
```

彼らは「固定された性質」ではなく「流れ」そのもの。
だから人型を保てない——形は固定されるものではなく、流れるものだから。

### なぜ封印されていたのか

玄冥が世界を「固定」した時、循環は不要になった。
「流れを止める」ことが玄冥の目的だったため、十二大従星は邪魔な存在だった。

| 時代 | 十二大従星の状態 |
|------|-----------------|
| 本来の世界 | 自由に存在し、世界の循環を支える |
| 玄冥の固定後 | 「循環」が否定され、封印される |
| 現在（天命院） | 十二司星官がその力を「借用」している |

玄冥が去った後、天命院は「固定された世界」を維持するため、十二大従星を封じ続けた。

### 刻夜にとっての意味

十大主星が「エンジン」なら、十二大従星は「燃料」。
どちらか一方だけでは、120干支を統べる力は発揮できない。

- 十大主星だけ：精神は強いが、力が足りない
- 十二大従星だけ：力はあるが、方向性がない
- **両方揃って初めて、玄冥と同等の力を得られる**

## 十大主星との違い

### 根本的な違い

| 項目 | 十大主星 | 十二大従星 |
|------|---------|-----------|
| **由来** | 十干（精神） | 十二支（時・エネルギー） |
| **姿** | 完全な人型 | 異形（流れは固定されない） |
| **性格** | 明確な自我 | 自我はあるが、より概念的 |
| **コミュニケーション** | 言葉で明確に会話 | 言葉より概念、感覚的 |
| **関係性** | 仲間、同志 | 従者、守護者 |

### なぜ十二大従星は異形なのか

十大主星は「固定された性質」——木・火・土・金・水という不変の概念。
だから人型という「固定された姿」を保てる。

十二大従星は「流れ」——胎から絶へと変化し続ける概念。
固定された姿を持つことは、彼らの本質に反する。
だから「異形」——人に近いが、人ではない。

### 関係性の違い：刻夜から見て

| 十大主星 | 十二大従星 |
|---------|-----------|
| **仲間、家族のような存在** | **従者、守護者のような存在** |
| 対等に会話し、意見を交わす | より献身的で、主に尽くす |
| 「俺たちは運命共同体」 | 「私たちは主のために存在する」 |
| 刻夜を「主」と呼ぶが、友のように接する | 刻夜を「主」と呼び、心からの崇敬を持つ |

### 十大主星と十二大従星の関係

互いを補完する存在として、協力関係にある。

| 十大主星から見た十二大従星 | 十二大従星から見た十大主星 |
|-------------------------|-------------------------|
| 「弟分」「妹分」のような存在 | 「兄」「姉」のような存在 |
| 守るべき存在、共に戦う仲間 | 尊敬する先輩、導いてくれる存在 |

```
例：
カンサク「お前たちも主の配下だ。胸を張れ」
スメラ「……我らが力を認めてくれるか、不動の者よ」

ギョクドウ「皆さん、刻夜様のお力になってくださいね」
カガミ「……はい、姉様」
```

## 刻夜との関係性

### 解放された後の関係

十二大従星は司星官を倒すことで解放される。
彼らにとって刻夜は「封印から救ってくれた恩人」であり「本来仕えるべき主」。

| 段階 | 十二大従星の態度 |
|------|-----------------|
| 解放直後 | 戸惑い。千年ぶりの「主」に困惑しつつも従う |
| 共に戦う中で | 刻夜の本質を理解し、心からの忠誠に変わる |
| 最終的に | 「この主のために存在できて良かった」という献身 |

### コミュニケーションの特徴

十大主星とは異なり、言葉より「感覚」「概念」で伝える傾向がある。

```
【十大主星との会話】
ギョクドウ「刻夜様、敵は三方向から来ます。私が東を、カンサクが西を」
刻夜「分かった。北は俺が抑える」

【十二大従星との会話】
シズメ「……主」（視線で敵を示す）
刻夜「……そうか。頼む」
シズメ「……」（頷き、影のように消える）
```

### 忠誠の表し方

十大主星が「共に歩む」なら、十二大従星は「支える」。

| 十大主星の忠誠 | 十二大従星の忠誠 |
|---------------|-----------------|
| 「俺はお前についていく」 | 「私はあなたのために存在する」 |
| 主の判断に意見を言う | 主の判断を信じて従う |
| 対等なパートナー | 献身的な従者 |

### 刻夜への影響

十二大従星と接することで、刻夜は「王」としての自覚を深める。

- 十大主星との関係：仲間として成長する
- 十二大従星との関係：王として責任を感じる

```
刻夜（十大主星に対して）「……助かった。次は俺が前に出る」
刻夜（十二大従星に対して）「……お前たちを、無駄死にはさせない」
```

### 冥界寄りの三体（シズメ・オサメ・サスラ）

封印されていないこの三体は、より深い絆を持つ可能性がある。

| 名前 | 刻夜との関係 |
|------|-------------|
| シズメ | 刻夜が冥界の力を使う時、最初に応える。「死」を恐れない強さを教える |
| オサメ | 刻夜の苦しみを「収めたい」と願う。刻夜の心の安息所 |
| サスラ | 刻夜の旅路を見守る。物語の終わりを知っている？ |

## ミナモト（天報星）
- **本質**: 胎
- **十二支**: 子（陽）
- **性別**: 男性的
- **忠誠の形**: **可能性** — 主の未来に無限の可能性を見出す。その可能性を開きたい
- **能力**: 可能性を操る。未来を揺らがせる
- **姿**:
  - 光る胞子のような無数の粒子が集合した人型
  - 輪郭が曖昧で、常に揺らいでいる
  - 声は複数の声が重なったように聞こえる
- **性格**:
  - 好奇心旺盛で純粋
  - まだ何者でもない「可能性」そのもの
  - 刻夜の選択に強い関心を持つ
- **口調**:
  - 「〜かもしれない」「〜という可能性が」
  - 断定を避ける話し方
- **封印者**: 十二司星官・子の司星官
- **登場予定**: 第三部

### 口調サンプル
```
「主よ、この先には無数の道がある」
「どの可能性を選びますか？」
「私は……まだ、何にでもなれる」
```

- **登場予定**: **第三部（冥界の干支を初使用する場面）**

### 口調サンプル
```
「……主。彼らを、終わらせましょうか」
「死は……静かなもの。怖くない」
「触れるだけ。それで、終わる」
```

### 能力詳細
- 触れた対象の時間を完全停止
- 冥界の力を使うため代償あり
- 最も「死」に近い存在

## オサメ ★（天庫星）【冥界寄り】
- **本質**: 墓
- **十二支**: 戌（陽）
- **性別**: 男性的
- **忠誠の形**: **安息** — 主に安らぎを与えたい。すべてを収める
- **能力**: 封印。あらゆるものを閉じ込める
- **姿**:
  - 地中に埋まった巨大な棺が人型を取ったもの
  - 重厚で威圧的な鎧のような外見
  - 目は閉じているが、すべてを見通す
- **性格**:
  - 静謐、終わりを司る
  - 穏やかだが、動くと止められない
  - 刻夜の苦しみを「収めたい」と願う
- **口調**:
  - 「収めよう」「眠れ」
  - 低く、厳かな声
- **封印者**: なし（冥界寄り）
- **登場予定**: **第三部以降**

### 口調サンプル
```
「主よ。その苦しみ……私が収めよう」
「この棺に……すべてを」
「眠れ。永遠に」
```

### 能力詳細
- 対象を「埋葬」する
- 一時的な無力化に最適
- 師匠の「封印」との対比になる

## サスラ ★（天馳星）【冥界寄り】
- **本質**: 霊
- **十二支**: 亥（陰）
- **性別**: 女性的
- **忠誠の形**: **回帰** — 主と共にすべての始まりへ。循環の終わりにして始まり
- **能力**: 遍在。同時に複数の場所に存在する
- **姿**:
  - 実体を持たない。風のように現れる
  - 時折、女性の輪郭が見える
  - どこにでもいて、どこにもいない
- **性格**:
  - 掴めない、神秘的
  - すべてを見通しているような達観
  - 刻夜の旅の終わりを知っている？
- **口調**:
  - 「私はここにいる……そこにも」
  - 複数の方向から同時に聞こえる
- **封印者**: なし（冥界寄り）
- **登場予定**: **第三部以降**

### 口調サンプル
```
「主……私はいつもあなたの傍に」
「すべては巡る。終わりは始まり」
「どこにいても……見ている」
```

### 能力詳細
- 同時多発攻撃
- 情報伝達にも使える
- 最も「霊的」な存在

## 登場タイミング

十二大従星は **第三部以降** に登場する。
序盤（第一部・第二部）は十大主星に集中し、キャラクターを覚えやすくする。

### 登場の流れ
1. **第三部**: 十二司星官を倒すたびに、封印されていた十二大従星が解放される
2. **冥界寄りの三体（シズメ・オサメ・サスラ）**: 刻夜が初めて冥界の干支を使用した際に顕現

### 冥界の三体について
封印されていないため、刻夜がいつでも呼べる。
しかし冥界の力を使うことになるため、代償がある。
