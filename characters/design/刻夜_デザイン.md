## 外見の詳細

### 髪

| 状態 | 詳細 |
|------|------|
| **覚醒前** | 黒髪。やや長めで、前髪が目にかかる程度 |
| **覚醒後** | 毛先に銀が混じり始める（冥界の力の影響）。物語が進むにつれ銀の割合が増加 |
| **スタイル** | 手入れされていない自然なまま。前髪は無造作に分けている |

### 瞳

| 状態 | 詳細 |
|------|------|
| **通常時** | 暗い灰色。光を反射しない、深く沈んだ色 |
| **「理を視る目」発動時** | 金色に変化。瞳の中に複雑な紋様が浮かぶ |
| **覚醒前** | 明るい灰色。柔らかな印象 |

### 顔立ち

- 整った顔立ちだが、頬がこけて不健康な印象
- 覚醒前は穏やかで柔らかい表情だったが、覚醒後は感情を見せない
- 眉は細め、目は切れ長
- 唇は薄く、あまり表情を作らない

### 体格・姿勢

- 骨格は標準的だが、肉がついていないため華奢に見える
- 姿勢は良いが、どこか力が抜けている
- 傷跡がリンチの痕として体の各所に残る

## 服装バリエーション

### 1. 序盤（追放直後）

```
【囚人服/ボロ着】
- 灰色のボロボロの囚人服
- 裾は破れ、泥と血で汚れている
- 裸足または粗末な草履
- 左手の甲の「空白の命刻」が露出
```

**デザインポイント**：
- 全体的に色褪せた灰色・茶色系
- 継ぎ接ぎだらけで、元の形が分からない
- 痩せ細った体がさらに強調される

### 2. 中盤（拠点確保後）

```
【黒を基調とした旅装】
- 黒のロングコート（膝丈）
- 黒のシャツとズボン
- 革のベルトとブーツ
- 左手には手袋（命刻を隠すため）
```

**デザインポイント**：
- シンプルで機能的
- 装飾は最小限
- 動きやすさ重視
- 全体的にダークトーン

### 3. 後半（「無刻の魔王」として）

```
【魔王の装束】
- 黒と銀の長衣（くるぶしまで届く）
- 襟は立てて、顔の下半分を隠せる
- 肩と袖口に冥界の紋様（薄く光る）
- 黒い手袋（左手の命刻を意図的に見せることも）
- 黒いブーツ
```

**デザインポイント**：
- 威圧感と神秘性
- 銀の差し色は髪の銀と呼応
- 紋様は冥界との繋がりを示す
- 全体的に「王」としての風格

## 特徴的なパーツ

### 空白の命刻（左手の甲）

```
【形状】
- 直径3cm程度の円形
- 文字や紋様は一切なし
- 薄く白銀に光る痕跡のみ

【状態変化】
- 通常時：うっすらと光る程度
- 力を使用時：強く輝く
- 冥界の力使用時：白銀から紫がかった光に変化
```

### 傷跡

```
【リンチの痕】
- 背中に鞭の痕（複数の線状）
- 腕に打撲痕
- 顔には目立つ傷なし（顔は避けられていた？）

【冥獣との戦闘痕】
- 首筋に噛み跡（覚醒時の傷）
- 薄くなっているが完全には消えていない
```

## デザイン上の注意点

1. **覚醒前後の対比を明確に**
   - 同一人物と分かりつつも、別人のような印象
   - 覚醒前：温かみ、柔らかさ
   - 覚醒後：冷たさ、鋭さ

2. **「空白」のイメージ**
   - 色彩を抑える（モノトーン中心）
   - 装飾を排した、シンプルなデザイン
   - 「何も持っていない」→「何にでもなれる」

3. **成長による変化**
   - 髪の銀の割合が増える
   - 服装が「囚人」→「旅人」→「王」へ
   - 表情の変化（完全な無表情→時折見せる感情）

4. **痩せた体格の表現**
   - 骨が浮き出ている
   - 服がややダボついている
   - しかし姿勢は良い（武人としての鍛錬）
