# 主人公

## 基本情報
- **名前**: 刻夜（コクヤ）
- **年齢**: 17歳
- **外見**:
  - **髪**: 黒髪。覚醒後、毛先に銀が混じり始める（冥界の力の影響）
  - **瞳**: 普段は暗い灰色。「理を視る目」発動時、金色に変わる
  - **体格**: 細身で華奢。リンチで痩せ細った身体
  - **服装**: ボロボロの囚人服から始まり、物語が進むにつれ黒を基調とした服装へ
  - **特徴**: 左手の甲に「空白の命刻」——命刻が刻まれるべき場所に、薄く光る円形の痕跡だけが残る（文字や紋様は一切なし）

### 覚醒前後の変化
| 覚醒前 | 覚醒後 |
|--------|--------|
| 穏やかな表情 | 感情を見せない |
| 明るい灰色の瞳 | 冷たい灰色、時に金 |
| 普通の黒髪 | 毛先に銀が混じる |

## 出自——玄冥の血筋

### 真実の血統

刻夜は**玄冥の直系の子孫**である。

| 項目 | 内容 |
|------|------|
| **血統** | 玄冥の妹・玄陽（げんよう）の末裔 |
| **両親** | 玄冥の血を引くが、自覚なく普通に暮らしていた |
| **誕生時** | 空白の命刻が現れ、両親は刻夜を守るために命を落とした |
| **発見者** | 師匠・星禍が刻夜を引き取り、封印して育てた |

### なぜ刻夜に空白が発現したか

- 千年の間、玄冥の血は薄まり「空白の因子」は眠っていた
- しかし世界の歪みが臨界点に達した今、因子が再燃した
- **玄冥の血 × 歪みの臨界点 = 空白として生まれる条件**

### 物語的な意味

- 刻夜は「玄冥のやり直し」——同じ血、同じ力
- しかし刻夜には仲間がいる。玄冥と違う選択ができる
- 血筋は呪いではなく、**可能性**

## 過去：失われた日常

- 両親を失い、師匠に引き取られた（本人は知らない）
- 養家族もでき、普通の命刻を持つ青年として暮らしていた
- 師匠を尊敬し、信頼していた

### 養家族（処刑された家族）

| 名前 | 役割 | 命刻 | 特徴 |
|------|------|------|------|
| **厳造（ゲンゾウ）** | 養父 | 甲寅（木の虎）| 木こり。寡黙で厳格だが情深い。主人公を「息子」と呼んだ |
| **紗代（サヨ）** | 養母 | 己未（土の羊）| 温和で優しい。孤児だった主人公を最初に受け入れた |
| **小雪（コユキ）** | 義妹 | 丁卯（火の兎）| 12歳。主人公を「兄ちゃん」と慕っていた。明るく活発 |

命刻の意味：
- **甲寅**：大木と虎。頑固で真っ直ぐな性格を表す
- **己未**：田園と羊。穏やかで家庭的な性格を表す
- **丁卯**：灯火と兎。繊細で可愛らしい、温かみのある性格を表す

### 転落

ある日、師匠が主人公の命刻を「消した」。理由も告げずに。

その瞬間からすべてが変わった：
- 「空白の命刻」の出現により、養家族が処刑された
- 仲間は離れ、村人は石を投げた
- 村から追放される
- 助けを求めても誰も取り合わない
- 守ってくれるはずの者も背を向けた
- **師匠は何も言わず、ただ見ていた**

## 覚醒

追放された直後、冥獣に襲われ瀕死の状態に。九死に一生の瞬間、能力が覚醒。

### 覚醒した能力
1. **理を視る目**: 時空の理を完全に把握する
   - 本質、相性、時機、因果、弱点——すべてが見える
2. **120干支へのアクセス**: 現世と冥界、両方の干支エネルギーを操れる可能性
3. **最初の配下**: ギョクドウ（十大主星）が覚醒と同時に現れる
   - 最初は一人だけ。力が足りないため
   - 復讐を重ねる中で残り九人が一斉に顕現

### 師匠が封じた理由
- 「空白の命刻」は災厄の象徴として恐れられている
- 発見されれば即座に処刑される——師匠はそれを避けるため封印した
- 普通の命刻を持つ子として育てることで、刻夜を守ろうとした

### 空白の真の意味（物語終盤で明かされる）
- 空白＝欠落ではなく「器」
- どの五行にも偏らない＝中庸
- 分断された現世と冥界を「繋ぐ」ことができる唯一の存在
- 玄冥と同じく、本来は調和をもたらす者だった

## 人物像

### 二面性
| 対象 | 態度 |
|------|------|
| 身内（配下） | 優しい。悩みを聞き、本質を肯定し、居場所を与える |
| 外敵 | 冷酷。命乞いも言い訳もすべて見透かし、一切の情けをかけない |

### 核となる信念
> 「この世界の理は歪んでいる。だから、俺が書き換える」

### 敵の定義——刻夜の「線引き」

刻夜には明確な「敵」の基準がある。

| 対象 | 判定 | 理由 |
|------|------|------|
| **天命院の構成員** | 殺す | 歪んだ理を維持する者 |
| **刻夜を殺そうとする者** | 殺す | 敵対行動を取った |
| **村人（追放した者）** | 殺さない | 殺す価値もない |
| **密告者** | 殺さない | 組織の一員ではない |
| **傍観者** | 殺さない | 眼中にない |

**刻夜の心理**：
```
「お前らを恨んでないわけじゃない」
「だが、お前らを殺しても何も変わらない」
「俺が壊すのは——この世界の理だ」
「雑魚に構っている暇はない」
```

→ **復讐ではなく革命**。個人への恨みより、システムへの怒り。

### 立ち位置
- **正義ではない**：救世主ではなく「矯正者」
- 善悪を超えた存在
- 世界からは「無刻の魔王」と呼ばれる

## 転落の詳細

### 第一段階：封印

ある夜、師匠が刻夜を呼び出した。

```
星禍「刻夜……一つ、頼みがある」
刻夜「何でしょう、先生」
星禍「……動くな」
（師匠の手が刻夜の胸に触れる。激痛）
刻夜「ぐっ……！　せ、先生……何を……！」
星禍「……すまない」
（意識が薄れていく）
刻夜「な……ぜ……」
星禍「これが、お前のためだと——信じたかった」
```

刻夜が目覚めた時、手の甲には「空白」の焼印が刻まれていた。

### 第二段階：発覚

翌朝、村の神殿で定期的な命刻検査が行われた。

```
下級司星官「次。名前は」
刻夜「刻夜です」
下級司星官「命刻を見せろ」
（刻夜の手の甲を確認した瞬間、司星官の顔が凍りつく）
下級司星官「……空白だと……？」
刻夜「え……？」
下級司星官「衛兵！　この者を拘束しろ！　災厄の印持ちだ！」
刻夜「待ってください！　昨日までは普通の——」
（混乱の中、刻夜は取り押さえられる）
```

### 第三段階：養家族の処刑

「空白を匿った罪」として、養家族は捕らえられた。

```
厳造「待ってくれ！　刻夜は昨日まで普通だった！　何かの間違いだ！」
司星官「命刻は嘘をつかない。この者は災厄の種だ」
紗代「刻夜は……刻夜は悪い子じゃありません……！」
小雪「兄ちゃん！　兄ちゃん！」
刻夜「やめろ……！　俺を殺せ！　家族は関係ない！」
司星官「黙れ、空白。お前が存在する限り、災いは続く」
（養家族は刻夜の目の前で処刑される）
刻夜「うああああああああ！」
```

### 第四段階：村人の豹変

処刑の後、村人たちの態度が一変した。

```
【太一】
太一「俺たち、ずっと災厄と一緒にいたのか……」
刻夜「太一……俺だよ、昨日まで一緒に——」
太一「近寄るな！　お前のせいで村に不幸が起きる！」

【美鈴】
刻夜「美鈴、頼む、俺の話を——」
美鈴「（目を逸らす）……ごめんなさい。私、何も知らない」
刻夜「美鈴……！」
美鈴「（逃げるように去る）」

【源次】
源次「お前を弟のように思っていた。だからこそ、許せない」
刻夜「源次兄……」
源次「お前がここにいたから、村に災いが来たんだ。厳造さんを、紗代さんを、小雪を——お前が殺したんだ」
刻夜「違う……！　俺は何もしてない……！」
```

### 第五段階：追放

村人たちは刻夜を村から追い出した。

```
【追放の描写】
村人A「災厄め！　ここから出て行け！」
（石が飛んでくる）
村人B「お前のせいで厳造さんが死んだんだ！」
太一「二度と戻ってくるな！」
（村の外へ追い出される）
刻夜「（村の門が閉まるのを見ながら）……師匠……」
（振り返る。師匠は遠くから見ているだけだった）
刻夜「……先生……」
星禍「……」（何も言わない。ただ、見ている）
刻夜「なぜ……なぜ何も言ってくれないんですか……」
```

### 第六段階：冥獣との遭遇

追放された刻夜は、森の中で冥獣に襲われる。

```
【森の中】
刻夜「（独白）……どこへ行けばいい」
刻夜「（独白）父さんも、母さんも、小雪も……もういない」
（突然、背後から殺気）
刻夜「！？」
（黒い霧を纏った獣——冥獣が現れる）
刻夜「な……何だ、これは……」
（冥獣が襲いかかる。刻夜は為す術もなく倒れる）
刻夜「（血まみれで）……俺は、何のために生まれてきたんだ」
（意識が薄れていく）
```

## 刻夜の選択肢

### 物語の分岐点

終盤、刻夜には三つの道が示される。

| 選択肢 | 内容 | 結末 |
|--------|------|------|
| **統べる** | 120干支をすべて統べる。最強になれるが、人間をやめる | 玄冥と同じ道。悲劇の繰り返し |
| **繋がる** | 配下との絆で力を「分かち合う」。人間のまま戦う | 新しい道。困難だが希望がある |
| **諦める** | 力を手放し、普通の人間として死ぬ | 平穏だが、世界は変わらない |

### 「繋がる」選択の意味

玄冥は「統べる」を選び、失敗した。
刻夜が選ぶべきは「繋がる」——仲間と共に力を分かち合うこと。

```
【最終決戦前】
ギョクドウ「主よ。120干支を統べれば、勝てます」
刻夜「……ああ」
ギョクドウ「しかし——」
刻夜「人間をやめることになる。分かっている」
カンサク「お前はどうしたい」
刻夜「……」
シャキ「俺たちはお前についていく。お前が人間をやめても」
ロクソン「でも……寂しいわ。今のあなたの方が好きだもの」
刻夜「……」

（長い沈黙）

刻夜「俺は——」
刻夜「お前たちと、一緒に戦いたい」
刻夜「一人で統べるんじゃない。みんなで、繋がる」
ギョクドウ「……主よ」
刻夜「それで負けるなら、それでいい」
刻夜「俺は、もう一人で戦うのは——嫌だ」

（十大主星、十二大従星、全員が刻夜を見る）

カンサク「……ったく。最初からそう言え」
シャキ「よし！　俺たちの力、全部使え！」
ギョクドウ「ふふ……やっと、本当の主になりましたね」
```

## 刻夜の成長アーク

### 第一部：復讐者

- 怒りに駆られ、すべてを壊そうとする
- 配下は「道具」に近い存在
- 目的は「天命院への復讐」

### 第二部：王

- 配下との絆が生まれる
- 「復讐」から「変革」へ目的がシフト
- しかし、冥界の力への依存が始まる

### 第三部：葛藤

- 人間性の喪失を自覚する
- 玄冥の過去を知り、自分の未来を恐れる
- 師匠との再会で、答えを探し始める

### 第四部：選択

- 「統べる」か「繋がる」かの選択
- 配下全員との絆を確認する
- 最終決戦へ
