# 勇者パーティ

天命の子・光輝に従う仲間たち。典型的な「勇者パーティ」構成だが、それぞれに深い背景と葛藤を持つ。

---

| 役割 | 名前 | 命刻 | 特徴 |
|------|------|------|------|
| 剣士（リーダー） | 光輝（コウキ） | 人工命刻 | 異世界転生者。主人公気取り |
| 聖女 | 白蓮（ビャクレン） | 癸卯 | 回復・浄化担当。光輝を盲信 |
| 騎士 | 鋼牙（コウガ） | 庚丑 | 前衛。光輝への忠誠は本物 |
| 魔術師 | 紫苑（シオン） | 丙午 | 攻撃魔法担当。やや懐疑的 |
| 斥候 | 疾風（ハヤテ） | 甲申 | 偵察・暗殺。任務優先 |

## 光輝（コウキ）【天命の子・リーダー】

### 基本情報
- **名前**: 光輝（コウキ）——神殿が与えた名。「光り輝く者」
- **前世の名**: 高橋翔太（たかはし しょうた）——平凡なサラリーマン
- **年齢**: 外見18歳（前世では32歳で死亡）
- **命刻**: 人工的に刻まれた「完璧な命刻」——偽物

### 外見
- 金髪碧眼。神々しいオーラ
- 白銀の鎧、聖剣を携える
- 「絵に描いたような勇者」
- 手の甲に「刻まれた完璧な命刻」——実は人工物

### 性格
- 本人に悪意はない（と本人は思っている）
- 天命に従って正しいことをしていると信じ込んでいる
- 「主人公であるべき」という自己認識
- 「主人公であるべき」という自己認識
- 実は前世でも「主人公」になれなかったコンプレックスがある
- **実績**: 数々の「災害級冥獣」を討伐し、民衆から英雄視されている

### 真実：異世界転生者
- **前世**: 現代日本で「なろう系」を読み漁っていた人間
- **転生の経緯**: 死後、この世界に転生。神殿に「天命の子」として見出された
- **偽りの命刻**: 元々は「命刻を持たない」異物だったが、神殿が人工的に「完璧な命刻」を刻んだ
- **チート能力**: 転生特典として与えられた力。この世界の理から外れた異質な力

### 口調例
```
「俺が来た！　もう大丈夫だ！」
「悪は必ず滅びる。それがこの世界の理だ」
「お前を倒せば、すべてが丸く収まる。……悪役は退場するのがお約束だろ？」
「（独白）やっと……やっと俺が主人公になれたんだ」
```

### 物語上の意味
- 主人公から見れば「作られた完璧さ」——文字通り「この世界のものではない」
- 民衆にとっては希望
- **「理を視る目」で本質が見える**: 刻夜だけが気づく違和感
- **テーマ対比**:
  - 天命の子 = 異世界から来た「主人公になろうとした者」
  - 刻夜 = この世界で「魔王にされた者」
  - どちらが本当の「主人公」か？

### 光輝の選択肢（物語分岐点）
刻夜との邂逅後、光輝には複数の道がある：
1. **否定して戦う**: 真実を認められず、刻夜と激突
2. **崩壊する**: アイデンティティを失い、暴走
3. **受け入れる**: 苦悩の末、自分の在り方を問い直す

## 白蓮（ビャクレン）【聖女】

### 基本情報
- **命刻**: 癸卯——水の兎。慈愛と献身、癒しの力
- **外見**: 銀髪、白いローブ。清楚な美少女
- **性格**: 純粋すぎて盲目。光輝を「救世主」と信じて疑わない
- **能力**: 回復魔法、浄化の光

### 出自
- **生まれ**: 黄土国の神殿都市。敬虔な信者の家庭
- **幼少期**: 神殿で「浄化の才能」を見出される
- **10歳**: 正式に聖女候補として神殿に入る
- **15歳**: 「天命の子」光輝の専属聖女に任命される

### 神殿での教育
```
【白蓮が教えられたこと】
- 「空白は災厄。見つけ次第浄化するのが聖女の務め」
- 「天命の子に仕えることが最高の栄誉」
- 「疑うことは罪。信じることが救い」
```

### 光輝への感情
- 最初は「任務」として仕えていた
- しかし光輝の真っ直ぐさ（表面上の）に惹かれていく
- 恋愛感情に近いが、それを「神聖な献身」と解釈している
- **本人は気づいていない** —— 信仰と恋を混同している

### 内面の葛藤
| 表面 | 内面 |
|------|------|
| 「光輝様は正しい」 | 時々、疑問が浮かぶ |
| 「空白は悪」 | 刻夜の目を見て動揺する |
| 「私は聖女」 | 私は何者なのか |

### 刻夜との対峙で起きること
- 白蓮は「浄化」しようとして刻夜に触れる
- しかし、何も起きない
- 「空白」なのに、悪意を感じない
- **聖女の力は「悪を浄化する」もの —— 刻夜が悪でないなら、効かない**
- これが白蓮の信仰を揺るがす最初のきっかけ

### セリフ例
```
【対刻夜・最初の対峙】
白蓮「我が光よ、この者の穢れを——」
（光が刻夜に触れるが、何も起きない）
白蓮「……なぜ？ 浄化されない……？」
刻夜「俺は穢れてなどいない。ただの人間だ」
白蓮「嘘です……空白は、空白は災厄の……」

【葛藤の深化】
白蓮「（独白）なぜ……あの者の目には、憎しみしかなかった」
白蓮「（独白）憎しみは……理由なく生まれない」
白蓮「（独白）私は……何を浄化しようとしていたのだろう」
```

### 物語上の役割
- 光輝の「正義」を支える存在
- 刻夜と対峙した時、最も葛藤する可能性

## 鋼牙（コウガ）【騎士】

### 基本情報
- **命刻**: 庚丑——金の牛。不動の精神と鋼の肉体
- **外見**: 巨漢の騎士。全身鎧
- **性格**: 愚直、忠義。光輝に命を救われた過去がある
- **能力**: 重装戦闘、盾術

### 出自
- **生まれ**: 白銀国の貧民街。孤児
- **幼少期**: 盗みを働いて生き延びる
- **12歳**: 盗みの最中に衛兵に追われ、絶体絶命
- **救済**: 偶然居合わせた光輝（転生直後）に助けられる

### 光輝との出会い
```
【12歳の鋼牙】
「死ぬ……俺はここで死ぬ」
（追い詰められた路地裏。衛兵の剣が振り下ろされる）
光輝「待て！　この子は俺が引き取る！」
鋼牙「……は？」
光輝「（笑顔）大丈夫。もう誰もお前を傷つけない」
（光輝はこの時、まだ「主人公」の役を演じることに酔っていた）
（しかし鋼牙にとっては、それが「本物の救い」だった）
```

### 忠誠の本質
- 鋼牙の忠誠は「光輝という人間」ではなく「あの瞬間」に向けられている
- 光輝が偽物だと知っても、動じない
- 「俺を救ったのは、あの時のあいつだ。今のあいつが何者でも関係ない」
- **忠誠とは何か、信じるとは何かを問う存在**

### 内面の単純さと深さ
| 単純さ | 深さ |
|--------|------|
| 「光輝様が言うなら正しい」 | 「正しいかどうかは俺が決める」 |
| 「敵は斬る」 | 「守るために斬る」 |
| 「理屈は分からん」 | 「理屈より大事なものがある」 |

### セリフ例
```
【光輝の真実を知った後】
紫苑「鋼牙、聞いたでしょう。光輝様の命刻は——」
鋼牙「ああ、聞いた」
紫苑「それでも……従うの？」
鋼牙「従う。理屈じゃねえんだ」
鋼牙「あの人は俺を救った。それは変わらん。たとえあの人が何者でも、俺の命は俺のもんだ。俺が誰に捧げるか決める」

【対刻夜】
鋼牙「お前は強いな。何のために戦っている？」
刻夜「……復讐だ」
鋼牙「復讐か。……分かる気がする」
刻夜「分かるものか」
鋼牙「俺も昔、すべてを奪われた。だから分かる。——でもな、復讐だけじゃ終わらんぞ」
```

### 物語上の役割
- 「信じる者」の象徴
- 光輝が偽物だと知っても動じない
- 刻夜と戦いながら、「こいつも何かを守っている」と感じる

## 紫苑（シオン）【魔術師】

### 基本情報
- **命刻**: 丙午——火の馬。情熱的だが自由奔放
- **外見**: 紫髪の女性魔術師。知的な眼鏡
- **性格**: 懐疑的、理性的。光輝の言動に違和感を覚えている
- **能力**: 火炎魔法、分析魔術

### 出自
- **生まれ**: 玄水国の学術都市。学者の家系
- **幼少期**: 神童と呼ばれる。魔術の才能は随一
- **14歳**: 天命院の研究機関に招聘される
- **16歳**: 「天命の子」のサポート役として派遣される

### 知識人としての懐疑
```
【紫苑の思考】
「データが合わない」
「天命の子の力は、この世界の理から外れている」
「命刻システムは……何かがおかしい」
（しかし証拠がない。だから黙っている）
```

### 光輝への感情
- 最初から「観察対象」として見ていた
- 光輝の力が「この世界のものではない」と直感的に気づいている
- しかし、光輝の純粋さ（表面上の）に情が移りつつある
- **「偽物でも本物になれるのか」という問いを持っている**

### パーティ内での役割
| 表向き | 本音 |
|--------|------|
| サポート魔術師 | 監視役（天命院から派遣） |
| 分析担当 | 光輝の「異常」を記録 |
| 中立的 | 最もまともな判断力を持つ |

### 分岐点
- 紫苑は物語中盤で「刻夜側に情報を流す」選択をする可能性がある
- 天命院への忠誠より、「真実」を優先する
- 最終的には中立か、刻夜側に寝返る

### セリフ例
```
【パーティ内での発言】
紫苑「光輝、ちょっと待って。……この村、妙に静かすぎない？」
光輝「大丈夫だって！　俺たちが来たんだから——」
紫苑「（ため息）……そういう話じゃないの」

【天命院への報告書（独白）】
紫苑「（書きながら）『天命の子の力は順調に成長中』……嘘。成長じゃない。暴走に近い」
紫苑「（書き直す）『異常なし』……これでいい。まだ、証拠がない」

【刻夜との対話】
紫苑「私は学者よ。真実が知りたいだけ」
刻夜「真実を知ってどうする」
紫苑「……分からない。でも、知らないままは嫌」
```

### 物語上の役割
- パーティ内の「良心」
- 刻夜側に寝返る可能性あり

## 疾風（ハヤテ）【斥候】

### 基本情報
- **命刻**: 甲申——木の猿。変幻自在な動きと道具使い
- **外見**: 小柄、フード姿。性別不明
- **性格**: 無感情、任務優先。誰に従うかは「勝つ側」
- **能力**: 隠密、暗殺、情報収集

### 出自
- **生まれ**: 不明。本人も覚えていない
- **幼少期**: 暗殺者ギルドに拾われ、道具として育てられる
- **訓練**: 感情を消す訓練を受ける。名前すら与えられなかった
- **現在**: 天命院に「貸し出された」最高位の暗殺者

### 「疾風」という名
- 天命院が与えたコードネーム
- 本人にとっては「名前」ではなく「記号」
- 呼ばれても反応するだけで、愛着はない

### 感情の欠如と残滓
| 表面 | 内面（本人も気づいていない） |
|------|----------------------------|
| 感情がない | かすかに「興味」がある |
| 任務が全て | 「なぜ」と問う自分がいる |
| 忠誠はない | 何かに属したい願望がある |

### 刻夜への反応
- 刻夜と対峙した時、初めて「殺したくない」と感じる
- 理由は分からない
- 刻夜の目に「同類」を見たから
- **疾風「……お前も、道具だったのか」**
- 刻夜「……違う。俺は道具になることを拒んだ」
- 疾風「……そうか」（初めて、何かを考える表情）

### 分岐点
- 疾風は「勝つ側につく」と公言している
- しかし、刻夜との接触で変わる可能性がある
- 最終的には「自分で選ぶ」ことを知る

### セリフ例
```
【任務中】
光輝「ハヤテ、敵の位置は？」
疾風「……三方向。北に五、東に三、南に七」
光輝「よし、俺が北を——」
疾風「（独白）……なぜ、一番多い方向に行く。非効率」

【刻夜との接触】
疾風「……依頼は、お前を殺すこと」
刻夜「……やってみろ」
（戦闘。疾風は刻夜を追い詰めるが、致命傷を与えない）
刻夜「……殺さないのか」
疾風「……分からない」
刻夜「分からない？」
疾風「今日は……気が乗らない」（去っていく）

【光輝の真実を知った後】
鋼牙「ハヤテ、お前はどうする」
疾風「……依頼主が変わるなら、依頼も変わる」
紫苑「つまり……？」
疾風「天命院が負けるなら、俺は天命院の敵になる。それだけ」
白蓮「そんな……信念はないの？」
疾風「……信念？」（首を傾げる。本当に分からない）
```

### 物語上の役割
- 「風見鶏」。状況次第で裏切る
- 刻夜との接触で「自分で選ぶ」ことを知る可能性

## 光輝パート（視点切り替え編）

物語の一部を光輝視点で描く。「勇者の冒険」として始まり、徐々に違和感が積み重なり、刻夜との邂逅で真実に至る。

### 構成

```
【光輝視点スタート】
「俺は天命の子。魔王を倒して世界を救う」
        ↓
【勇者として活躍】
民を救い、称賛される日々
        ↓
【四神との遭遇】
守護神のはずが、なぜか光輝を拒絶する
        ↓
【違和感の蓄積】
・四神が「世界の理に反する者」と呼ぶ
・チート能力が時々暴走する
・紫苑が「おかしい」と指摘し始める
        ↓
【刻夜との邂逅】
「理を視る目」で真実を暴かれる
        ↓
【世界観崩壊】
自分が「この世界の異物」だと知る
```

### 四神が光輝を拒絶する理由

四神は騰蛇に操られているが、「世界の理」に対する本能は残っている。

- 光輝は「この世界の理から外れた存在」
- 命刻を持たずに生まれた異物
- 四神は本能的に「排除すべき」と感じる
- 騰蛇の支配下でも、この本能だけは抑えきれない
- **光輝への拒絶反応が、四神が「操られている」ことの伏線になる**

### 刻夜との邂逅シーン

```
光輝「ついに見つけたぞ、無刻の魔王！　俺がお前を——」
刻夜「……」
光輝「何だ、その目は。怖いか？　俺は天命の子だ。完璧な命刻を——」
刻夜「お前の命刻……存在しない」
光輝「は？」
刻夜「いや、違う。書き込まれている。後から。……偽物だ」
光輝「嘘だ……嘘だ！　俺は選ばれた！　この世界の——」
刻夜「この世界の者ではない。お前は——異物だ」
```

### 光輝視点での伏線

| 場面 | 光輝の解釈 | 真実 |
|------|-----------|------|
| 四神が冷淡 | 「試練か？」 | 異物だと見抜いている |
| 力が暴走 | 「まだ制御できてないだけ」 | この世界の理と相容れない |
| 民が少し怯える | 「魔王の噂のせいだ」 | 本能的に「違う」と感じている |
| 紫苑の懐疑 | 「心配性だな」 | 唯一まともに分析している |

## 戦闘能力

### パーティ全体の戦力
- 十大主星＋十二大従星が必要な相手
- 勇者パーティ単独で下級司星官を圧倒可能
- 十大主星＋十二大従星が必要な相手
- 勇者パーティ単独で下級司星官を圧倒可能
- 十二司星官と互角以上に戦える
- **対冥獣**: 災害級冥獣を討伐できる唯一のパーティ（光輝のチート火力による）

### 個別戦力

| メンバー | 戦闘スタイル | 危険度 |
|----------|-------------|--------|
| 光輝 | チート能力による圧倒的火力 | 最高（暴走時） |
| 白蓮 | 回復・支援・浄化 | 中（攻撃は苦手） |
| 鋼牙 | 重装戦闘・盾役 | 高（持久戦に強い） |
| 紫苑 | 火炎魔法・分析 | 高（遠距離攻撃） |
| 疾風 | 暗殺・急所攻撃 | 最高（一撃必殺） |
