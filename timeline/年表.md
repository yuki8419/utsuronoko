# 年表

世界の歴史から物語の時系列までを網羅。

---

## 創世〜神話時代

### 創世の時代（時期不明）

| 出来事 | 詳細 |
|--------|------|
| 世界の誕生 | 現世と冥界は一つだった——あるいは、境界が曖昧だった |
| 120干支の成立 | 現世の60干支と冥界の60干支が循環 |
| 十大主星・十二大従星の誕生 | 十干・十二支の「概念」が人格を持った存在として顕現 |
| 四神＋騰蛇の配置 | 「世界の楔」として東西南北＋中央に配置 |
| 空白の存在 | 「調停者」として自然に存在し、両界の調和を保つ |
| 天命院の成立 | 空白と協力し、命刻の管理と人々の生活を支える組織として設立 |

### 創世後〜玄冥以前（詳細不明）

- 現世と冥界が対等に存在し、行き来可能
- 命刻は「傾向」であり、固定された運命ではない
- 空白は稀な存在になりつつあったが、まだ「調停者」として機能
- **空白と天命院の協力関係**により世界の調和が保たれていた
- 大司星は空白と天命院を繋ぐ役割を担っていた
- 五つの大国（蒼龍国、朱炎国、黄土国、白銀国、玄水国）の原型が形成

### 空白発生間隔の崩壊——なぜ法則が乱れたか

**200年周期の崩壊**
```
千年前 → 800年前 → 600年前 → 400年前 → 200年前 → 50年前 → 17年前
      200年    200年     200年     200年     150年    33年
                                              ↑        ↑
                                           縮小開始  加速
```

**原因：歪みの臨界点**
- 玄冥の「固定」から千年、歪みが限界まで蓄積
- ダムに例えれば「亀裂の連続発生」状態
- 空白の発生＝歪みの「ガス抜き」だが、もはや追いつかない
- 刻夜が覚醒まで生き延びた＝世界が「調停者」を求めている
- このまま放置すれば、空白の発生間隔はさらに縮まり、やがて世界は自壊する

| 変化 | 詳細 |

| 命刻 | 「固定」から「循環」に戻る。傾向であり運命ではない |
