# 小説プロジェクト「虚ろの子」

## プロジェクト概要

「空白の命刻」を持つ主人公・刻夜が、すべてを奪われた絶望から覚醒し、歪んだ世界の理を書き換えていく物語。陰陽五行をベースにした命刻システムが特徴。

## 作業別の参照ファイル

### 執筆時
1. **必須**: plot/guides/第X章.md（該当章のガイド）
2. 執筆ガイドは自己完結型。基本的にそのファイルだけで執筆可能

### 執筆ガイド作成時
**必ずタスク化すること**。以下のステップを順に実行：

1. [ ] settings/配下設定.md「本編タイムライン」を確認
2. [ ] 状態スナップショットを記入（テンプレート参照）
3. [ ] 構成提案（話数・タイトル・内容）→ **ユーザー承認待ち**
4. [ ] 承認後、ガイド本文作成（plot/guides/_テンプレート.md に沿う）
5. [ ] 整合性チェック（禁止事項・伏線・タイムラインとの矛盾）

※構成提案の承認なしにガイド作成に入らないこと
※タスク化を怠ると矛盾が発生しやすい

### 設定追加・修正時
1. rules/整合性ルール.md のセクション12を参照
2. 必ずSSOTファイルに追加/修正
3. 派生ファイルがあれば同時に更新

### 矛盾チェック時
1. rules/整合性ルール.md の全セクションを確認
2. 特に注意：玄冥≠刻夜（別人物）

## 重要な設定ポイント

### 玄冥と刻夜の違い
| | 玄冥（千年前） | 刻夜（現在） |
|---|---|---|
| 封印 | なし（最初から空白露見） | あり（17年間隠されていた） |
| 選択 | 「統べる」 | 「繋ぐ」を選べる |

### 情報開示タイミング
- 玄冥の名前 → 第五部まで隠す
- 刻夜が玄冥の血筋 → 第三部〜第四部
- 星禍の真意 → 第四部

詳細は rules/整合性ルール.md セクション8を参照。
