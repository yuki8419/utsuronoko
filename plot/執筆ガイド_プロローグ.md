# 執筆ガイド：プロローグ「虚ろの子」

## この章の目標

1. 千年前の「空白」の悲劇を断片的に描き、物語の重層性を示す
2. 刻夜と同じ道を歩んだ「先駆者」の存在を印象づける
3. 「統べる」という選択の悲しさを伝える
4. 読者に「この人は誰なのか」という謎を与える

---

## 物語上の位置づけ

- **視点人物**：玄冥（ただし名前は明かさない）
- **時代**：千年前
- **機能**：刻夜の物語との対比、伏線の設置、世界観の提示

---

## 真実と「隠す情報」

### ★重要：プロローグで意図的に隠す真実

プロローグでは以下の情報を**絶対に明かさない**。これらは終盤で明かすことで、物語に大きなどんでん返しを生む。

| 隠す情報 | 真実 | 明かすタイミング |
|---------|------|----------------|
| **大司星の存在** | 玄冥を利用して「固定」させた裏切り者 | 第五部・萬星戦前後 |
| **十大主星・十二大従星の反対** | 彼らは「固定」に反対したが、絶対服従のため従った | 第四部・騰蛇戦あたり |
| **天命院との協力関係** | 本来は空白と天命院は協力関係だった | 第三部～第四部で徐々に |
| **玄冥の名前** | 「玄冥」という名 | 第五部・冥界での対話 |

### プロローグでの描き方

```
【真実】
大司星が玄冥に近づき、信頼を得て、「固定」の方法を教えた
            ↓
【プロローグでの描写】
孤独の果てに、自ら「統べる」ことを選んだように見せる
```

## シーン構成と演出指針

### シーン1：幸せだった頃（回想）

**目的**：読者の感情移入、「失われた日常」の美しさ

- 具体的な名前・顔は出さない
- 温かさ、匂い、音で情景を伝える
- 「どこにでもある幸福」を強調

```
【OK】
「笑い声」「温かい食卓」「眩しい笑顔」
→ 普遍的で、刻夜の日常とも重なる

【NG】
「○○という村で」「△△という名の女が」
→ 具体的すぎる
```

### シーン2：拒絶の記憶

**目的**：空白への迫害の普遍性を示す、刻夜の未来を暗示

- 「虚ろの子」という呼称を使う
- 石を投げる群衆→顔のない悪意
- **最も信じた人に目を逸らされる**→最大の絶望

```
【演出ポイント】
- 群衆は「声と影」だけ
- 個人の顔は描かない（読者が想像できるように）
- 「愛した人」は性別・関係を明示しない
```

### シーン3：力の覚醒と孤独

**目的**：十大主星・十二大従星の存在を示唆、「統べる」選択への道筋

#### ★注意：本来の真実との整合性

**真実**：
- 玄冥は力を得た後も優しすぎて復讐できなかった
- 迫害は続いた
- 大司星が近づき、「固定」を教えた

**プロローグでの描写**：
- 力を得た瞬間から「統べる」を選んだように見せる
- 大司星の存在は完全に隠す
- 十大主星・十二大従星が「反対した」ことは描かない

```
【現在の本編の描写】（これで正しい）
「彼らが何者なのか、俺にはわかった」
「彼らは何も言わない。ただ、じっと俺を見ている。待っている」
→ 中立的に見ている描写。反対していたことは隠している
```

### シーン4：世界の固定

**目的**：「固定」の瞬間を描く、玄冥の善意と悲しみを示す

- 玄冥の涙を描く→悪意ではなく、善意からの行動
- 「二度と俺のような者は生まれない」→自己犠牲的な思い
- 壮大だが、どこか空虚な描写

```
【演出ポイント】
- 力の描写は抽象的に（具体的な術名は出さない）
- 「すまない」「これしかできなかった」→後悔と諦念
- 最後に見たのは「失われた日常」→本当は繋がりたかった
```

### シーン5：千年後への橋渡し

**目的**：現在の刻夜に繋げる、問いかけで終わる

- 玄冥の声で語る（ナレーション形式）
- 「お前は、どうする？」→選択の余地があることを示す
- 「俺にはできなかった」→繋がることへの未練

## 伏線管理

### この章で張る伏線

| 伏線 | 描写 | 回収タイミング |
|------|------|---------------|
| 「虚ろの子」という呼び名 | 玄冥が呼ばれる | 刻夜も同じように呼ばれる |
| 「繋がりたかった」 | 玄冥の本心 | 玄冥との対話で回収 |
| 「俺と同じ道を歩くか」 | 問いかけ | 最終章で刻夜が答えを出す |
| 「俺にはできなかった」 | 繋がることへの未練 | 刻夜が「繋ぐ」を選ぶ時の対比 |
| 涙を流しながらの「固定」 | 善意からの行動 | 真実が明かされた時、さらに悲劇性が増す |

### ★隠された真実（回収時のインパクトのため）

| 隠された伏線 | プロローグの印象 | 真実が明かされた時 |
|-------------|----------------|-------------------|
| 大司星の裏切り | 玄冥が自ら選んだ | 「騙されていた」→悲劇性が倍増 |
| 十大主星たちの反対 | 彼らは黙って見ていた | 「止めたかったのに従うしかなかった」→彼らの苦しみ |
| 天命院との協力関係 | 天命院は後から設立 | 「元々は味方だった」→現在の天命院の堕落が際立つ |

## 文体・口調

### 玄冥の一人称

- 「俺」で統一
- 短い文、断片的な描写
- 感情を抑えた、乾いた語り口
- ただし要所で感情が溢れる

```
【基本トーン】
淡々と、過去を振り返るように。
けれど時々、抑えきれない感情が漏れる。

【例】
「俺は独りだったんだな」
「最初から、ずっと」
→ 短い文の連続で、痛みを表現
```

### 避けるべき表現

- 具体的な地名・人名
- 詳細な戦闘描写
- 設定の説明的な文章
- 十大主星・十二大従星の固有名

---

## 読者への効果（狙い）

1. **謎**：「この人は誰？」「いつの話？」
2. **予感**：「主人公も同じ道を歩むのでは」
3. **緊張感**：「この日常が壊されるのだろう」
4. **共感**：「繋がりたかった」という普遍的な願い
5. **期待**：「主人公は違う選択ができるのか」

### ★終盤での効果

真実が明かされた時：
- 玄冥への印象が「世界を歪めた加害者」→「利用された被害者」へ反転
- 天命院（萬星）への怒りが増す
- 刻夜が「繋ぐ」を選ぶことの意味が深まる
