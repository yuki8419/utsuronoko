# 第2章改訂方針メモ

**作成日**: 2025-12-05
**ステータス**: 確定

---

## 議論の背景

第2章の後半（第4話・第5話）において、以下の問題点が指摘された：

1. **リンチ描写が過度に暗い**
   - 村人による集団リンチは読者に不快感を与えすぎる
   - 「ダークだけどカッコいい」というコンセプトから逸脱

2. **村人への復讐が適切か**
   - 村人を蹂躙対象にすると、刻夜が単なる加害者に見える
   - オーバーロード/無限ガチャのような「筋が通った悪」にならない

3. **覚醒のトリガーが弱い**
   - 人間による暴力より、もっと「世界のシステム」に関わる脅威が適切

---

## 決定事項

### 1. 村人の扱い

**変更前**: 村人がリンチ → 刻夜が復讐
**変更後**: 村人は追放のみ → 刻夜は村人を「超越」する

**理由**:
- 村人は「理」（天命院のシステム）の被害者でもある
- 刻夜が真に戦うべきは「理そのもの」であり、末端の村人ではない
- 「お前たちなど、もう眼中にない」という態度がカッコいい

### 2. 覚醒トリガーの変更

**変更前**: リンチで瀕死 → 覚醒
**変更後**: 追放後に冥獣に襲われる → 覚醒

**理由**:
- 冥獣は「固定の代償」＝世界のシステムが生んだ脅威
- 刻夜が戦うべき敵を明確化
- 世界設定（冥獣）の早期導入で伏線効果も

### 3. 刻夜の動機

**変更前**: 村人への復讐
**変更後**: 真実を知る → システムへの反逆

**流れ**:
1. 養家族を殺したのは「空白を匿った罪」という「理」
2. その「理」を作ったのは天命院
3. 天命院が広める「運命は変えられない」を打ち砕く

### 4. テーマの明確化

| 対立軸 | 天命院 | 刻夜 |
|--------|--------|------|
| 主張 | 運命は変えられない | 運命は変えられる |
| 根拠 | 固定された命刻システム | 空白＝システム外の存在 |
| 目的 | 支配の維持 | 理の書き換え |

**世界設定との整合性**:
- `世界設定.md` の記述：「本来の状態：傾向、可能性。変化しうる」「固定後：絶対的な運命。逃れられない」
- 玄冥が世界を「固定」したことで運命が絶対になった
- 刻夜はこれを「元に戻す」＝運命は変えられる、と主張する

## スタイルガイド

### 刻夜の方向性

**「筋が通った悪」のポイント**:
- 敵には容赦しない（天命院、司星官）
- 味方には優しい（十大主星、配下）
- 無関係な者は殺さない（村人は追放で終わり）
- 理由なき暴力はしない

**カッコいい台詞の方向性**:
```
「お前たちを恨んでいない。恨む価値もない」
「俺が壊すのは、お前たちではなく——この世界の理だ」
「運命？　そんなものは、俺が書き換える」
```

### オーバーロード/無限ガチャ的な要素

- 圧倒的な力の差を見せる（覚醒後）
- 敵の卑劣さを先に描く（天命院の非道）
- 主人公側の「正当性」を読者に納得させる
- 配下との絆を丁寧に描く
