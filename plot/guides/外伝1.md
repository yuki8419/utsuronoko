## この外伝の目標

1. 「拠点」の確立——生活基盤が整う第一歩
2. 刻夜の初めての決断「ここで生きる」を描く
3. **顕現までの「溜め」**——3回失敗し、4回目で成功
4. 刻夜の変化——「一人で生きる」から「助けを求める」へ
5. シロク顕現＋侍従24人による拠点整備
6. シロクの力の凄さを見せる——刻夜が10日かけてもできなかったことを数分で

## 各話のプロット

### 第1話「廃墟」（2,500〜3,500字）

**目的**：廃砦発見。刻夜の「ここで生きる」という初めての決断。しかし顕現には至らない

**シーン構成**：

1. **森の放浪**
   - 場所：深い森の中
   - 時間：第3章終了から数日後・昼
   - 内容：
     - 刻夜とギョクドウが森を歩いている
     - 刻夜はまだ無気力。ギョクドウの後をついていくだけ
     - 時折立ち止まる。足が重い
     - ギョクドウ「……少し、休みましょうか」
     - 刻夜「……いい。歩ける」——強がり

2. **影衆の報告**
   - 影衆が戻ってくる（影から現れる）
   - ギョクドウ「……見つけたのですね」
   - 影衆は言葉を発さない。存在するだけで意思が伝わる
   - 「廃砦がある。冥獣被害で放棄された監視塔」
   - ギョクドウ「刻夜様、拠点になりえる場所が見つかりました」

3. **廃砦へ**
   - 場所：森の奥、丘の上
   - 廃砦に到着。荒れ果てた石造りの塔
   - 壁は崩れ、屋根は穴だらけ、蔦が絡みついている
   - とても人が住める状態ではない
   - 刻夜「……これが、拠点か」
   - ギョクドウ「私の眷属だけでは修繕は難しいですが……」

4. **刻夜の決断**
   - 刻夜は廃砦を見上げる
   - （心の声）「……ここに、住むのか」
   - （心の声）「逃げ続けるのか。それとも——」
   - ギョクドウ「他を探すこともできます。刻夜様の意思に従います」
   - 長い沈黙
   - 刻夜「……ここでいい」
   - 刻夜「**ここで、生きる**」
   - 初めて自分の意思で決断した——その重さ
   - ギョクドウ「（微かに微笑んで）……承知いたしました」

5. **顕現の兆し——そして失敗**
   - 決断した瞬間、刻夜の手の甲が**微かに光る**
   - ギョクドウ「……！」
   - しかし、光はすぐに消えてしまう
   - ギョクドウ「……惜しい。届きかけましたが……」
   - 刻夜「……何だ、今のは」
   - ギョクドウ「次の星が目覚めかけたのです。ですが——まだ、足りないようです」
   - 刻夜「……足りない？」
   - ギョクドウ「言葉だけでは、届かないのかもしれません」

**ポイント**：
- 刻夜はまだ受動的。ギョクドウについていくだけ
- 廃砦は「住めない状態」として描写
- 「ここで生きる」が初めての能動的な決断
- **しかし、それだけでは顕現しない**——「言葉だけでは足りない」という布石
- 読者に「何が足りないのか？」という興味を持たせる

### 第2話「限界」（2,500〜3,500字）

**目的**：自分たちだけで生活を作ろうとするが、限界を知る

**シーン構成**：

1. **翌朝——整備開始**
   - 場所：廃砦の中庭
   - 時間：1話から翌日・朝
   - ギョクドウ「まずは住める状態にしましょう」
   - 影衆12体を呼び出す
   - しかし影衆は偵察・情報収集が専門。掃除や修繕は不得手
   - 瓦礫を動かそうとするが、実体が薄くて持てない
   - ギョクドウ「……彼らは影。物を運ぶのは難しいのです」

2. **刻夜も手伝う**
   - 刻夜が瓦礫を運ぼうとする
   - しかし体がまだ回復していない。すぐに息が切れる
   - ギョクドウ「刻夜様、無理をなさらず……」
   - 刻夜「……黙ってろ」
   - 意地になって続ける。だが、ほとんど進まない
   - 一つ瓦礫を動かすのに、何分もかかる

3. **一日の終わり——ほとんど変わらない**
   - 場所：廃砦の入口
   - 時間：夕方
   - 一日かけて——廃砦はほとんど変わっていない
   - 瓦礫を少し片付けただけ。壁の穴はそのまま。屋根は崩れたまま
   - 刻夜は疲れ果てて座り込んでいる
   - ギョクドウが水を差し出す
   - 刻夜「……こんなものか」
   - ギョクドウ「私の眷属では、生活を支えることは難しいのです」
   - 刻夜「……」

4. **夜——「生きる」ことの重さ**
   - 場所：廃砦の一角（野宿に近い状態）
   - 屋根のある場所を探して寝る。地面は固い
   - 刻夜は眠れない
   - （心の声）「『ここで生きる』と言った」
   - （心の声）「だが——生きるとは、こういうことか」
   - （心の声）「食事を作る。寝る場所を整える。水を汲む。火を起こす」
   - （心の声）「……俺は、何も知らなかった」
   - 師匠の家では、すべてが当たり前にあった
   - 食事は出てきた。布団はあった。屋根があった
   - 今は——何もない
   - 刻夜「……生きるって、こんなに面倒なのか」

5. **ギョクドウとの会話**
   - ギョクドウ「眠れませんか」
   - 刻夜「……ああ」
   - ギョクドウ「焦らなくても良いのです。時間はあります」
   - 刻夜「……時間があっても、俺たちだけじゃ無理だ」
   - ギョクドウ「……」
   - 刻夜「お前の影は戦えるが、掃除はできない。俺は——体が動かない」
   - ギョクドウ「だからこそ、次の星が必要なのです」
   - 刻夜「次の星……」
   - ギョクドウ「生活を支える者。あなたがそれを本当に求めれば——」
   - 刻夜「求める……？」
   - ギョクドウ「言葉ではなく、心で。本当に『生きる場所を作りたい』と——」

**ポイント**：
- 「生活を作る」ことの難しさを体感させる
- 刻夜が「生きること」に本気で向き合い始める
- ギョクドウだけでは限界があることを示す
- 「言葉ではなく、心で求める」というヒントを出す

### 第3話「献身」（2,500〜3,500字）

**目的**：刻夜が「生きる場所を作る」ことに本気で向き合う。光が強くなるが——まだ届かない

**シーン構成**：

1. **十日目の朝**
   - 場所：廃砦
   - 時間：廃砦発見から10日後
   - 刻夜が起き上がる。体が痛い
   - 毎日の作業で全身が悲鳴を上げている
   - だが——また瓦礫に向かう
   - 刻夜「……やるか」

2. **黙々と働く**
   - 刻夜は瓦礫を運ぶ
   - 少しだけ慣れた。でも遅い
   - 影衆も手伝う。できることを探している
   - ギョクドウも手を動かす
   - 誰も何も言わない。ただ、動く
   - 10日かけて——少しは変わった。床の瓦礫は半分ほど片付いた
   - それでも——まだ住める状態には程遠い

3. **刻夜の心の変化**
   - 刻夜は手を動かしながら考える
   - （心の声）「師匠は——一人で俺を育てた」
   - （心の声）「飯を作って、服を繕って、薬を調合して」
   - （心の声）「あれは——当たり前じゃなかったんだな」
   - （心の声）「生きるって——誰かが支えてくれていたから、できていたことだ」
   - ギョクドウとの会話——千年待っていた
   - 待っていてくれた者たちがいる

4. **決意——本当の意味での「生きる」**
   - 刻夜「……俺は、ここで——生きたい」
   - 刻夜「お前たちと——生きる場所を、作りたい」
   - 今度は——心からそう思った

5. **光が強くなる——しかし消える**
   - 刻夜の手の甲が光る——今度は強い光
   - 腕を伝い、広がっていく
   - ギョクドウ「来る——」
   - だが——光は、また消えた
   - ギョクドウ「先日より、ずっと強い光でした」
   - ギョクドウ「近づいています。確実に」
   - 「いい」は諦め。「生きたい」は願い——その違い

6. **夜——諦めない**
   - 刻夜は眠れないが、心は少し軽い
   - 願うこと。生きる場所を作りたいと願うこと
   - まだ答えは出ない
   - でも——諦めるつもりはなかった
   - もう一度——試してみよう。明日——また

**ポイント**：
- 「生きること」への本気の向き合いが顕現の鍵
- 1話の決断との違いを明確に——言葉だけ vs 心から
- **光は強くなるが、まだ届かない**——次話への布石
- 3回目の挑戦として「近づいている」ことを示す

### 第4話「顕現」（3,000〜4,000字）

**目的**：雨がトリガーとなり、「助けてくれ」という祈りでシロク顕現。侍従による整備と刻夜の驚き

**シーン構成**：

1. **十一日目の朝——雨**
   - 場所：廃砦
   - 時間：11日目の朝
   - 刻夜は雨音で目を覚ます
   - 天井から水が滴っている
   - 外は土砂降り。風も強い
   - 床が水浸しになっている
   - 10日かけて片付けた床が、泥水で汚れている

2. **雨の中の絶望**
   - 屋根のある場所がほとんどない
   - 逃げ場がなくなっていく
   - 午後になっても雨は止まない
   - 刻夜の怒りと絶望
   - 刻夜「……くそっ」
   - 10日間の作業が——たった半日の雨で台無しに

3. **限界**
   - 刻夜「俺たちだけじゃ——無理だ」
   - 人手がいる。道具がいる。材料がいる
   - 何もかもが足りない
   - 濡れた床に座り込む刻夜

4. **助けを求める**
   - 刻夜「……誰か」
   - 刻夜「誰か——助けてくれ」
   - 生まれて初めて、他者に助けを求めた
   - それは——祈りだった

5. **シロク顕現**
   - 刻夜の手が光る——これまでにない強い光
   - 全身を包み込む
   - ギョクドウ「来ます——！」
   - 光から女性が現れる
   - 灰色の髪、眼鏡、地味な服装、帳簿を持つ
   - シロク「司禄星——シロクと申します」
   - シロク「何かお役に立てることはございますか」

6. **侍従24人の顕現と整備**
   - シロクが侍従24人を呼び出す
   - 信じられないほど速い動き
   - 床の水を掻き出す。屋根の穴を塞ぐ。壁を補修する
   - 刻夜が10日かけて運んだ瓦礫が——あっという間に片付く
   - 一刻も経たないうちに「住める場所」に

7. **刻夜の驚き**
   - 部屋に案内される
   - 布が敷かれ、火が焚かれている。暖かい
   - 刻夜「……すごいな」
   - 「俺たちが10日かけてもできなかったことを——一刻で」
   - ギョクドウ「これが——司禄星の力です」
   - 刻夜、初めて誰かに助けを求め、助けが来た
   - 雨音を聞きながら眠りに落ちる

**ポイント**：
- 雨がトリガー——自然の力で追い詰められる
- 「助けてくれ」という祈りが顕現の鍵
- 3回失敗した後の4回目の成功——溜めが効いている
- シロクの力の凄さを視覚的に見せる
- 顕現後の「ありがとう」や疎外感は描かない——顕現＋整備＋驚きで終了

## 文体・トーンの指針

### この外伝のトーン
- 静かで穏やか。だが刻夜の心には居場所のなさがある
- シロクの献身は「押しつけ」に近く見える瞬間がある
- 3話目で「気づき」があり、温かさが生まれる
- 「生活が始まる」希望の光

### 文体の特徴
- 一人称/三人称：三人称（刻夜視点中心）
- 文の長さ：短めの文を基調。決断・気づきの瞬間は特に短く
- ダッシュ「——」の使用：感情の揺れや重要な瞬間に
- 独白の頻度：刻夜の内面は3話目で増える（気づきの過程）

### 文体サンプル

**【疎外感のシーン】の書き方**：
```
　侍従たちが動き回っている。

　瓦礫を運ぶ者。床を掃く者。壁を補修する者。

　刻夜は——その中に、自分の居場所がなかった。

「……俺は、何をすればいい」

　ギョクドウが振り返った。

「お休みになってください。体を休めることも大事です」

「……」

　休む？　俺だけが？

　皆が働いている中で、俺だけが何もしない？

　それは——どこか、座りが悪かった。
```

**【気づきのシーン】の書き方**：
```
「……お役に立てましたでしょうか」

　シロクの声が、微かに震えていた。

　刻夜は——その声に、引っかかりを感じた。

　報告じゃない。確認でもない。
　この女は——何かを待っている。

　何を、だ？

　刻夜は、部屋を見回した。

　清潔な床。整えられた寝台。窓に張られた布。

　昨日まで廃墟だった場所が——今は、住める場所になっている。

　誰がやった？
　この女だ。

　なぜ？
　俺のためだ。

　……ああ、そうか。
　こいつは——認められたいんだ。
```
