## 各話のプロット

### 第1話「目覚め」（3,000字）

**目的**：安全な場所での目覚め、ギョクドウとの出会い、刻夜の混乱と警戒

**シーン構成**：

1. **洞窟での目覚め**
   - 場所：森の奥、隠れた洞窟
   - 時間：夜明け前
   - 内容：
     - 刻夜が意識を取り戻す
     - 焚き火の暖かさ、体の痛み
     - 傍らに座る人影——長い黒髪の女性
     - 「お目覚めですか」穏やかな声

2. **ギョクドウの第一印象**
   - 冷たい美貌だが、刻夜を見る目だけが柔らかい
   - 「……お体の具合はいかがですか」
   - 刻夜の混乱（見知らぬ女性、記憶の断片）

3. **記憶の蘇り**
   - 処刑、追放、冥獣——断片的に蘇る
   - 「お前は……誰だ」
   - 「あの化け物を……殺したのか」
   - ギョクドウ「はい。あなたを傷つけようとした者を、排除いたしました」
   - 刻夜は返事をせず、再び眠りに落ちる（体力の限界）

**ポイント**：
- ギョクドウの自己紹介は最小限（「私はギョクドウと申します」のみ）
- 刻夜は質問する気力もない状態
- 「保護されている」感覚と「何も分からない」不安の両方を描く

### 第2話「拒絶」（3,000字）

**目的**：刻夜の無気力と人間不信、ギョクドウへの拒絶

**シーン構成**：

1. **数日後の朝**
   - 刻夜は目を覚ますが、起き上がろうとしない
   - ギョクドウが食事を運んでくる
   - 刻夜「……いらない」
   - 「食べなければ、死にますよ」
   - 「……それでいい」

2. **無気力**
   - 家族の顔が浮かぶ。小雪の声が聞こえる気がする
   - 「お兄ちゃん、約束だよ？」
   - ——もう、守れない
   - 刻夜は壁を向いたまま動かない

3. **ギョクドウの問いかけ**
   - 「……なぜ、死にたいのですか」
   - 刻夜「……」
   - 「生きる理由がないから？　それとも——」
   - 刻夜「うるさい。放っておいてくれ」
   - ギョクドウ「……かしこまりました」
   - 静かに距離を取るギョクドウ。しかし洞窟を去りはしない

4. **夜の独白**
   - 刻夜は眠れない
   - （なぜ、あの女は俺のそばにいる）
   - （何が目的だ。俺に何をさせたい）
   - 信じることができない。誰も。何も

**ポイント**：
- 刻夜の「生きる意味の喪失」を丁寧に描く
- ギョクドウは押し付けない。待つ姿勢
- 人間不信（太一たちに裏切られた直後）を強調

### 第3話「復讐の炎」（3,500字）

**目的**：復讐衝動の噴出、ギョクドウによる制止、復讐の虚しさを予感

**シーン構成**：

1. **悪夢**
   - 刻夜は夢を見る
   - 処刑台。父の首。母の悲鳴。小雪の——
   - 叫びながら目を覚ます

2. **怒りの爆発**
   - 「殺してやる……」
   - 声が漏れる。抑えられない
   - 「太一を。美鈴を。源次を。全員——」
   - 立ち上がろうとする。足がもつれる

3. **洞窟を出ようとする刻夜**
   - 「村に戻る」
   - ギョクドウ「……お体が万全ではありません」
   - 「関係ない。あいつらを殺す。それだけだ」
   - よろめきながら歩き出す刻夜

4. **ギョクドウの制止**
   - ギョクドウが刻夜の前に立つ
   - 「お止めください」
   - 「どけ」
   - 「止めません」
   - 刻夜がギョクドウを殴ろうとする。ギョクドウは避けない
   - 拳が止まる。——殴れなかった

5. **問いかけ**
   - ギョクドウ「村人を殺して、どうなりますか」
   - 刻夜「……気が済む」
   - 「本当に？」
   - 「……」
   - 「村人を殺しても、他の村は変わりません。天命院も変わりません」
   - 刻夜「じゃあ……俺はどうすればいい」
   - 膝が折れる。座り込む刻夜

**ポイント**：
- 復讐に走ろうとする「ワンクッション」
- ギョクドウは止めるが、説教はしない
- 刻夜自身が「殴れなかった」ことで、復讐の虚しさを予感する
- まだ答えは出ない

### 第4話「十大主星」（3,000字）

**目的**：ギョクドウの正体の一部を明かす。ただし詳細は保留

**シーン構成**：

1. **落ち着いた朝**
   - 前話から数日後
   - 刻夜は食事を取るようになった（最低限）
   - まだ無口だが、ギョクドウの存在を拒絶しなくなった

2. **刻夜からの問いかけ**
   - 「……お前は、何者だ」
   - 初めて刻夜から話しかけた
   - ギョクドウ「お話しする準備が、できましたか」
   - 「……聞くだけだ」

3. **最低限の説明**
   - ギョクドウ「私は『十大主星』の一人です」
   - 「十大主星……？」
   - 「あなたの命刻に眠っていた——十の存在」
   - 刻夜「命刻に……眠っていた？　俺は『空白』だ。何もないはずだ」
   - ギョクドウ「……いずれ、詳しくお話しします。今は——」

4. **保留される情報**
   - 刻夜「なぜ、俺の中にいた」
   - ギョクドウ「……それを説明するには、長い話が必要です」
   - 「今はまだ、その時ではありません」
   - 刻夜「……」
   - 不満そうだが、追及する気力もない

5. **他の九人**
   - 「他の九人は？」
   - ギョクドウ「まだ、眠っております」
   - 「私が最初に目覚めたのは、あなたの覚醒が引き金でした」
   - 「しかし、他の者を目覚めさせるには——まだ、力が足りません」
   - 刻夜「今は……お前だけか」
   - ギョクドウ「はい。ですが、私だけでもあなたをお守りすることはできます」

**ポイント**：
- 「十大主星」「命刻に眠っていた」という事実だけを伝える
- 「なぜ空白に」「千年前に何があったか」は**保留**
- 情報を出し惜しみすることで、読者の興味を引く

### 第5話「空白」（3,000字）

**目的**：空白の命刻の可能性を示唆（断定しない）、世界の歪みを匂わせる

**シーン構成**：

1. **洞窟の外へ**
   - 刻夜が初めて洞窟の外に出る
   - 森の空気。鳥の声。——平穏な世界
   - だが刻夜には、何も感じない

2. **空白についての対話**
   - 刻夜「……空白の命刻は、災厄だと言われた」
   - ギョクドウ「ええ。この世界では、そう教えられています」
   - 「お前は……どう思う」
   - ギョクドウ「私は——」
   - 少し間を置いて
   - 「空白は『欠陥』ではないと、考えております」

3. **可能性としての空白**
   - ギョクドウ「何も刻まれていない。それは——何でも受け入れられる、ということかもしれません」
   - 刻夜「かもしれない……？」
   - ギョクドウ「確かなことは、私にも分かりません」
   - 「ただ——あなたには、可能性がある。それだけは、信じております」
   - 断定を避ける。「器」という言葉も使わない

4. **世界の歪みへの言及**
   - 刻夜「なぜ……空白は排除される」
   - ギョクドウ「この世界の命刻は——少し、歪んでいるのです」
   - 「本来、命刻はもっと——」
   - 言葉を切る
   - 「……いずれ、お話しします」
   - 刻夜「また『いずれ』か」
   - ギョクドウ「申し訳ありません。今は——まず、生きることを」

5. **家族への想い**
   - 刻夜は黙って森を見つめる
   - （父さん。母さん。小雪——）
   - 「……俺は、何をすればいい」
   - ギョクドウ「まずは——生き延びることです」
   - 刻夜「……そうか」
   - 大きな決意ではない。ただ、「死なない」という選択

**ポイント**：
- 「器」「変革」という大きな概念は出さない
- 空白の意味は**示唆**に留める
- 世界の歪みの原因（千年前の固定化）は**言及しない**
- 刻夜の決意は「生き延びる」レベル（壮大な目標はまだ）

### 第6話「最初の一歩」（3,500字）

**目的**：小さな決意、理を視る目の片鱗（偶発的発動）、次章への引き

**シーン構成**：

1. **小雪の夢**
   - 刻夜は夢を見る
   - 小雪が笑っている
   - 「お兄ちゃん、頑張ってね」
   - 目が覚める。涙が流れている

2. **決意——とまではいかないが**
   - 刻夜は洞窟の入り口で朝日を見る
   - 「……生きる」
   - 声に出してみる
   - 「生きて……それから考える」
   - ギョクドウが傍に来る
   - 「まずは、安全な場所を探しましょう。ここは長く留まれません」

3. **行動の開始**
   - 刻夜「……ああ」
   - 初めて、前向きな返事
   - ギョクドウの表情が、わずかに和らぐ
   - 「私の——眷属が、偵察に出ております」
   - 刻夜「眷属……？」
   - ギョクドウ「いずれ、お見せします」

4. **理を視る目——偶発的発動**
   - 森を歩いている時、刻夜の目が一瞬金色に光る
   - 世界が——違って見えた
   - 木々の中を流れる何か。光の筋。生命の脈動
   - 「——っ」
   - 頭痛。すぐに消える
   - ギョクドウ「……刻夜様？」
   - 刻夜「……いや。何でもない」
   - （今のは……何だ）
   - 自分でも分からない。説明もできない

5. **章の終わり**
   - 二人は森の中を歩いていく
   - 刻夜「……ギョクドウ」
   - 「はい」
   - 「……ありがとう」
   - 小さな声。振り返らずに言った
   - ギョクドウ「……」
   - 一瞬、ギョクドウの目が揺れる
   - 「——こちらこそ」
   - 朝日の中、二人は歩き出す
   - まだ何も分からない。何も決まっていない
   - だが——一人ではない
   - それだけが、今は確かだった

**ポイント**：
- 「理を視る目」は一瞬だけ、偶発的に発動
- 刻夜自身も何が起きたか分からない（説明しない）
- 本格的な能力覚醒は外伝以降
- 刻夜の「ありがとう」でギョクドウの感情が揺れる（伏線）
- 感謝を伝える区切りの話

### 第7話「道標」（3,000字）

**目的**：今後の方針を示す、第4章への橋渡し

**シーン構成**：

1. **森の中での会話**
   - 二人で森を歩いている
   - 刻夜「……これから、どうする」
   - 初めて刻夜から「今後」を問いかける
   - ギョクドウ「まず——安全な場所を確保する必要があります」

2. **現状の整理**
   - ギョクドウ「あなたの存在は、おそらく天命院に伝わっています」
   - 刻夜「追手が来る、か」
   - ギョクドウ「はい。遠からず」
   - 「この森も長くは安全ではありません」

3. **眷属の存在を示す**
   - ギョクドウ「私には——眷属がおります」
   - 刻夜「眷属……？」
   - ギョクドウが手を掲げると、木々の影から黒い霧が滲み出る
   - 12の人影が現れ、音もなく跪く
   - ギョクドウ「影衆。偵察と情報収集を担う者たちです」
   - 刻夜「……」（驚くが、もう何が起きても不思議ではないという顔）

4. **当面の方針**
   - ギョクドウ「影衆に周囲を探らせます。拠点になりそうな場所を」
   - 「廃村、廃砦——人目につかず、守りやすい場所」
   - 刻夜「俺は……何をすればいい」
   - ギョクドウ「まずは体を休めてください。戦いはこれからです」
   - 刻夜「……戦い、か」
   - ギョクドウ「あなたが望むなら——私はどこまでもお供いたします」

5. **章の終わり**
   - 影衆が散っていく。音もなく、森に溶けるように
   - 刻夜「……分かった」
   - 「お前に任せる」
   - ギョクドウ「……お任せください」
   - 微かに微笑むギョクドウ
   - 二人は歩き続ける。行く先は分からない。だが——道はある
   - それだけで、今は十分だった

**ポイント**：
- 「これからどうするか」を具体的に示す
- 眷属（影衆）の初登場——軍勢の片鱗
- 刻夜は受け身だが、「任せる」と委ねる＝信頼の芽生え
- 第4章（集結）への自然な橋渡し
