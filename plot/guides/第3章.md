## 各話のプロット

### 第1話「目覚め」（3,000〜4,000字）

**目的**：安全な場所での目覚め、ギョクドウの第一印象（慈愛）、冥獣殲滅との対比

**シーン構成**：

1. **洞窟での目覚め**
   - 場所：森の奥、隠れた洞窟
   - 時間：夜明け前
   - 内容：
     - 刻夜が意識を取り戻す
     - 焚き火の暖かさ、柔らかい布の感触
     - 傍らに座る人影——長い黒髪の女性
     - 「お目覚めですか、刻夜様」穏やかな声

2. **ギョクドウの第一印象**
   - 冷たい美貌だが、刻夜を見る目だけが柔らかい
   - 冥獣を屠った冷徹さは微塵も見せない
   - 「……お体の具合はいかがですか」
   - 刻夜の混乱と警戒（見知らぬ女性、記憶の混乱）

3. **刻夜の混乱**
   - 記憶が断片的に蘇る——処刑、追放、冥獣、そして……
   - 「お前は、誰だ」
   - 「あの化け物を……殺したのか」
   - ギョクドウ「はい。あなたを傷つけようとした者を、排除いたしました」
   - その言葉の冷たさと、表情の柔らかさの対比

**ポイント**：
- ギョクドウの「二面性」を匂わせる（慈愛と冷徹）
- 刻夜が警戒しつつも、どこか安心している描写
- 環境描写で「保護されている」感覚を出す

### 第2話「十大主星」（3,000〜4,000字）

**目的**：ギョクドウの正体、十大主星の存在を提示

**シーン構成**：

1. **自己紹介**
   - ギョクドウ「私はギョクドウ……あなたの命刻に眠っていた者です」
   - 刻夜「命刻に……眠っていた？」
   - 「空白」の命刻から何かが生まれるという矛盾への疑問

2. **十大主星の説明**
   - 「私を含め、十の存在があなたの内に眠っておりました」
   - 十干の概念が人格を持った存在
   - 刻夜の覚醒と共に、順次顕現していく
   - ギョクドウ「私は最初に目覚めました。あなたをお守りするために」

3. **「なぜ俺に」という疑問**
   - 刻夜「俺は……空白だ。何も持っていない」
   - ギョクドウ「いいえ。空白だからこそ、私たちは眠っていられたのです」
   - 空白＝何も刻まれていない＝何でも受け入れられる「器」
   - 刻夜はまだ納得していないが、話を聞く姿勢に

4. **他の主星への言及**
   - 「他の九人は？」
   - ギョクドウ「まだ、眠っております」
   - 「私が最初に目覚めたのは、あなたの覚醒が引き金でした。しかし、他の九人を目覚めさせるには、まだ力が足りません」
   - 刻夜「今は……お前だけか」
   - ギョクドウ「はい。しかし、私だけでもあなたをお守りすることはできます」

**ポイント**：
- 設定説明を会話の中で自然に
- 刻夜の疑問＝読者の疑問として処理
- 「力が増すにつれ順次顕現」で外伝への伏線

### 第3話「空白の意味」（3,000〜4,000字）

**目的**：空白の命刻の真の意味、世界の理の歪みを提示

**シーン構成**：

1. **世界の理の説明**
   - ギョクドウが命刻システムについて説明
   - 生まれた時に刻まれる「魂の刻印」
   - 五行属性、性格傾向、運命——すべてが決まる
   - 刻夜「それで俺は……『空白』だから排除されるのか」

2. **空白の真の意味**
   - ギョクドウ「空白は『欠陥』ではありません」
   - 何も刻まれていない＝どの五行にも偏らない中庸
   - 分断されたものを繋ぐ存在——それが「器」の本質
   - 刻夜「繋ぐ……？」

3. **世界の歪み**
   - ギョクドウ「この世界の命刻は——歪んでいます」
   - 本来は「傾向」を示すものが、「逃れられない運命」になっている
   - 冥獣が溢れ出すのも、この歪みが原因
   - 誰かが命刻を「固定」した——千年前に

4. **家族を殺したもの**
   - 刻夜「……俺の家族を殺したのは、誰だ」
   - ギョクドウ「村人です。検査官です。天命院です」
   - 「しかし——彼らもまた、この理に縛られている」
   - 「あなたの家族を殺したのは、村人個人ではない。この世界の理です」
   - 刻夜の表情が変わる——怒りの対象が明確になる

**ポイント**：
- 復讐対象を「村人」から「世界の理」へ転換する重要な話
- 設定説明だが、刻夜の感情を軸に進める
- ギョクドウの「千年前」への言及で伏線

### 第4話「決意」（3,000〜4,000字）

**目的**：刻夜の決意、ギョクドウとの信頼構築

**シーン構成**：

1. **刻夜の葛藤**
   - 一人で考える時間
   - 養家族の顔、小雪との約束、失われた日常
   - 怒りはある——だが、誰に向けるべきか
   - 村人か？　天命院か？　それとも——

2. **理への反逆**
   - 刻夜「……俺は、この理を書き換える」
   - 村人を殺しても何も変わらない
   - 天命院を潰しても、また同じことが起きる
   - 変えるなら、根本から——世界の理そのものを

3. **ギョクドウの反応**
   - 「それこそが——私たちが待ち望んだ言葉です」
   - 一瞬、目に涙のような光——すぐに消える
   - 刻夜「……待っていた？」
   - ギョクドウ「……千年。私たちは、あなたを待っていました」

4. **主従の誓い**
   - ギョクドウが跪く
   - 「私の智、私の策、私のすべてを——あなたに捧げます」
   - 刻夜「……俺は、まだ何も成し遂げていない」
   - ギョクドウ「だからこそ。共に歩ませてください」
   - 刻夜がギョクドウの手を取る——最初の絆

**ポイント**：
- 刻夜の決意は「復讐」ではなく「変革」
- ギョクドウの「千年」発言で考察要素を仕込む
- 感動的なシーンだが、ベタになりすぎない

### 第5話「理を視る目」（3,000〜4,000字）

**目的**：能力の開眼、次章への引き

**シーン構成**：

1. **力の自覚**
   - 決意した刻夜の体に変化
   - 手の甲の「空白」が淡く光る
   - ギョクドウ「始まりました……あなたの力が、目覚めます」

2. **「理を視る目」の開眼**
   - 刻夜の瞳が金色に変わる
   - 世界が——違って見える
   - 木々の命刻、風の流れ、時間の脈動
   - 「視える……すべてが」

3. **能力の説明**
   - ギョクドウ「『理を視る目』……世界の本質を見通す力です」
   - 本質、相性、時機、因果、弱点——すべてが視える
   - 「あなたは『空白』だからこそ、すべてを視ることができる」
   - どの属性にも偏らないから、すべてを等しく視られる

4. **ギョクドウを「視る」**
   - 刻夜がギョクドウを視る
   - 水の属性、深い知性、そして——抑え込まれた感情
   - 「お前は……」
   - ギョクドウ「（少し動揺）……何か、視えましたか」
   - 刻夜「……いや。何でもない」
   - 二人の間に、新しい何かが生まれる

5. **次章への引き**
   - 「次に目覚めるのは、誰だ」
   - ギョクドウ「あなたの状況次第です。まず必要なもの——安全な場所、食事、休息」
   - 「眷属」への言及（次章以降への伏線）
   - 夜明け、二人で新しい一日を始める
   - ※まだ他の主星は顕現していない。外伝で順次顕現

**ポイント**：
- 能力獲得のカタルシス
- ギョクドウの感情を「視てしまう」ことで関係に変化
- 「まずは生き延びる」という現実的な課題を提示

## 文体・トーンの指針

### この章のトーン
- 静かな始まり、徐々に高まる感情
- 説明的なシーンが多いが、感情を軸に進める
- 最後は希望と決意で締める

### 文体の特徴
- 一人称/三人称：三人称（刻夜視点中心）
- 文の長さ：説明シーンは長め、感情シーンは短く
- ダッシュ「——」の使用：感情の断絶、重要な言葉の前に
- 独白の頻度：刻夜の内面描写は多め

### 文体サンプル

**【静かな対話シーン】の書き方**：
```
　ギョクドウが語る。声は穏やかで、しかし底に何かを秘めている。
　刻夜は黙って聞いていた。理解しようとしている——いや、理解したくないのかもしれない。
　「……つまり、俺は」
　「はい。あなたは——」
　言葉が途切れる。ギョクドウの目が、一瞬だけ揺れた。
```

**【決意のシーン】の書き方**：
```
　刻夜は立ち上がった。
　足が震えている。体が重い。それでも——立った。
　「俺は、この理を書き換える」
　声は小さかった。けれど、確かだった。
　ギョクドウの瞳が見開かれる。そして——
　「それこそが」
```
