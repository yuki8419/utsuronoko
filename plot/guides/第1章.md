## この章の目標

1. 刻夜の「失われる前の日常」を描き、読者に感情移入させる
2. 養家族との絆、師匠への信頼を丁寧に描写する
3. 師匠・星禍による封印解除を描く
4. 「空白の命刻」という概念と、それがいかに恐れられているかを示す
5. プロローグ（玄冥）と対比させ、「同じ悲劇の始まり」を予感させる

---

## 前章までのあらすじ

### 読者が知っていること（プロローグの内容）
- 千年前、「虚ろの子」と呼ばれた青年が迫害され、絶望の果てに世界を「固定」した
- 命刻システムが生まれ、「空白」は二度と生まれないはずだった
- しかし千年後、また一人の「空白の者」が生まれてしまった
- 「お前は、どうする？」という問いかけで終わっている

### 執筆者用メモ（読者は知らない）
- プロローグの「俺」は玄冥（名前は終盤まで明かさない）
- 刻夜は17年前、玄冥の妹・玄陽の末裔として生まれた
- 生まれた瞬間「空白の命刻」が現れ、両親は刻夜を守るため命を落とした
- 師匠・星禍が刻夜を引き取り、封印して「普通の子」として育てた
- 刻夜はこれらの事実を知らない
- 玄冥と刻夜は**別人物**（血縁関係はあるが同一人物ではない）

## 各話のプロット

### 第1話「平穏な日々」（3,000〜4,000字）

**目的**：読者に「失われる幸福」を見せる

**シーン構成**：
1. **夕暮れの帰り道**
   - 場所：蒼龍国辺境の村への道
   - 時間：夕方
   - 内容：稽古帰りの刻夜。緑豊かな森、穏やかな空気。「ここが俺の居場所だ」という静かな幸福感

2. **養家族との夕食**
   - 場所：刻夜の家
   - 時間：夕方〜夜
   - 内容：厳造、紗代、小雪との温かいやり取り。笑い声に満ちた食卓

3. **就寝前のひととき**
   - 場所：縁側、寝室
   - 時間：夜
   - 内容：小雪との会話「兄ちゃん、ずっとここにいてね」。刻夜の穏やかな返事

**ポイント**：
- プロローグ①の「笑い声が聞こえる」と対比
- 温かさを丁寧に描くことで、後の喪失感を際立たせる
- 命刻の話題は出さない（日常の一部として当たり前）

### 第2話「師との稽古」（3,000〜4,000字）

**目的**：師匠との関係と、不穏な予兆を示す

**シーン構成**：
1. **朝の稽古**
   - 場所：村外れの稽古場
   - 時間：早朝
   - 内容：木剣での打ち合い。「構えが甘い。もう一度」という厳しい指導

2. **休憩中の会話**
   - 場所：稽古場の木陰
   - 時間：午前
   - 内容：「先生、俺……いつか先生みたいに強くなれますか」→星禍「お前は、強くならなくていい」

3. **稽古の終わりと村での日常**
   - 場所：稽古場→村
   - 時間：午前〜昼
   - 内容：星禍が刻夜の手の甲を一瞬見つめる。村では太一・美鈴との交流

**ポイント**：
- 星禍の「お前は、強くならなくていい」は重要な伏線
- 師匠への尊敬と信頼を描く
- 小さな違和感を読者に感じさせる

### 第3話「小さな幸福」（3,000〜4,000字）

**目的**：日常の愛おしさを最大限に描く（嵐の前の静けさ）

**シーン構成**：
1. **午後の手伝い**
   - 場所：森（木こり仕事）
   - 時間：午後
   - 内容：厳造との会話。「お前は……立派になった」「お前は……俺の誇りだ」

2. **紗代との会話**
   - 場所：家の台所
   - 時間：夕方
   - 内容：夕食準備を手伝う。刻夜が孤児だったことを匂わせる

3. **小雪との約束**
   - 場所：縁側
   - 時間：夜
   - 内容：「明日、森の奥に連れてって！」→刻夜が約束する。**この約束は果たされない**

**ポイント**：
- 各キャラクターとの絆を丁寧に描写
- 「幸せな日常」の頂点
- 小雪との約束が果たされないことで悲劇性を強調

---

### 第4話「呼び出し」（3,000〜4,000字）

**目的**：不穏な空気の高まりと、運命の夜への導入

**シーン構成**：
1. **その日の異変**
   - 場所：稽古場→村
   - 時間：朝〜夕方
   - 内容：師匠が稽古場に来ない。一日中姿を見せない不安

2. **いつもと変わらない夕食**
   - 場所：家
   - 時間：夕方〜夜
   - 内容：表面上は普通だが、刻夜は何かがおかしいと感じる

3. **深夜の呼び出し**
   - 場所：家→森への道
   - 時間：深夜
   - 内容：星禍が現れ「……来い。話がある」。月明かりの下を歩く二人

**ポイント**：
- 緊張感を高める
- 星禍の内面の葛藤を「沈黙」で表現
- 森の描写で不穏さを演出

---

### 第5話「封印解除」（3,000〜4,000字）

**目的**：日常の終わり、悲劇の始まり

**シーン構成**：
1. **森の奥**
   - 場所：森の開けた場所
   - 時間：深夜
   - 内容：月光の下、星禍が刻夜の前に立つ

2. **封印解除**
   - 場所：同上
   - 時間：深夜
   - 内容：「刻夜……一つ、頼みがある」「動くな」→師匠の手が刻夜の手の甲に触れる→激痛

3. **真実の一端と暗転**
   - 場所：同上
   - 時間：深夜
   - 内容：「これが、お前のためだと——信じたかった」→意識を失う刻夜。翌朝は定期命刻検査

**ポイント**：
- 星禍の動機は謎のまま（読者の疑問を残す）
- 「空白の命刻」の恐ろしさを示唆
- 第2章への強い引き

| 用語 | 説明（この章で必要な範囲） |

```
左手の甲に「空白の命刻」——命刻が刻まれるべき場所に、
薄く光る円形の痕跡だけが残る（文字や紋様は一切なし）
```

## 文体・トーンの指針

### この章のトーン
- **第1〜3話**：温かさ、穏やかさ、日常の幸福
- **第4話**：不穏さ、緊張感の高まり
- **第5話**：衝撃、悲劇の始まり

### 文体の特徴
- 一人称/三人称：三人称（刻夜視点）
- 文の長さ：短文と長文のリズム
- ダッシュ「——」の使用：効果的に使用（感情の高まり、余韻）
- 独白の頻度：刻夜の内省を適度に挟む

### 文体サンプル

**【日常の描写】**：
```
夕陽が森を染めていた。

稽古を終えた帰り道、刻夜は深く息を吸った。土と緑の匂い。遠くで鳥が鳴いている。

この村で生まれ育ったわけではない。けれど今は——ここが、俺の居場所だ。

煙突から煙が上がっているのが見える。紗代が夕食を作っているのだろう。
今日の献立は何だろうか。そんなことを考えながら、刻夜は足を速めた。
```

**【緊張の描写】**：
```
「……来い」

星禍の声は、いつもより低かった。

刻夜は頷き、寝床から身を起こした。養家族を起こさないよう、音を立てずに戸を開ける。

外は月明かりに照らされていた。銀色の光が、師匠の横顔を浮かび上がらせる。

——何かが、おかしい。

言葉にはできない違和感が、胸の奥でざわめいていた。
```

**【衝撃の描写】**：
```
「動くな」

師匠の手が、刻夜の左手を取った。
そして——手の甲に、掌を当てた。

「……っ！」

瞬間、激痛が走った。

焼けるような痛み。皮膚を剥がされるような痛み。骨まで染み渡る、凄絶な痛み。

「ぐっ……！　せ、先生……何を……！」
```
