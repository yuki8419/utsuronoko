## この章の目標

1. 「空白の命刻」発覚による恐怖と混乱を描く
2. 養家族の処刑シーンで読者に衝撃と悲しみを与える
3. 村人の豹変——昨日までの「仲間」が「敵」になる絶望を描く
4. 村からの追放と冥獣の襲撃で、覚醒への土台を作る
5. プロローグの玄冥と同じ「すべてを奪われる」体験を刻夜にさせる
6. 刻夜が戦うべき敵は「村人」ではなく「世界の理」であることを示唆する

## 前章までのあらすじ

### 第1章で起きたこと（読者が知っている）
- 刻夜は蒼龍国辺境の村で、養家族（厳造・紗代・小雪）と平和に暮らしていた
- 師匠・星禍との稽古。「お前は、強くならなくていい」という謎の言葉
- 小雪と「明日、森の奥に連れていく」と約束した
- その夜、星禍に森に呼び出され、封印を解除された
- 刻夜の手の甲に「空白の命刻」が現れ、意識を失った
- 翌朝は定期命刻検査の日

### 執筆者用メモ（読者は知らない）
- 星禍が封印を解除した理由は謎のまま（第四部で明かす）
- 養家族は刻夜が「空白」だとは知らなかった（封印されていたため）
- 太一・美鈴・源次は第1章で「友人」として描写済み。この章で裏切る
- 刻夜はまだ「玄冥の血筋」であることを知らない

## 各話のプロット

### 第1話「発覚」（実績：約2,500字）

**目的**：空白発覚の衝撃と、パニックの始まり

**シーン構成**：
1. **目覚め**
   - 場所：森の中
   - 時間：早朝
   - 内容：刻夜が意識を取り戻す。体が重い。手の甲を見ると——薄く光る円形の痕跡。何も刻まれていない

2. **村への帰還**
   - 場所：村への道→村
   - 時間：朝
   - 内容：何が起きたのか理解できないまま村に戻る。村は定期命刻検査で騒がしい

3. **検査官との遭遇**
   - 場所：村の広場
   - 時間：朝
   - 内容：天命院の下級司星官が検査中。刻夜の命刻を見た瞬間、顔が凍りつく。「空白だと……？」

**ポイント**：
- 刻夜の混乱と恐怖を丁寧に描く
- 検査官の反応で「空白」の恐ろしさを示す
- 村人たちのパニックが始まる

### 第2話「断罪」（実績：約2,600字）

**目的**：「空白を匿った罪」による養家族の拘束

**シーン構成**：
1. **拘束**
   - 場所：村の広場
   - 時間：朝
   - 内容：刻夜が捕らえられる。「災厄の印持ちだ！」という叫び

2. **養家族の連行**
   - 場所：刻夜の家→広場
   - 時間：朝〜昼
   - 内容：「空白を匿った罪」で厳造・紗代・小雪も連行される。彼らは何が起きているのか分からない

3. **弁明の拒否**
   - 場所：広場
   - 時間：昼
   - 内容：厳造「刻夜は昨日まで普通だった！　何かの間違いだ！」→司星官「命刻は嘘をつかない」

**ポイント**：
- 養家族が「知らなかった」ことを強調
- 刻夜の「俺を殺せ！　家族は関係ない！」という叫び
- 理不尽な断罪の恐怖

### 第3話「処刑」（実績：約2,600字）

**目的**：養家族の処刑。この物語で最も悲惨なシーン

**シーン構成**：
1. **処刑宣告**
   - 場所：村の広場
   - 時間：昼
   - 内容：「空白を匿い、災厄を村に招いた罪」で死刑宣告

2. **最後の言葉**
   - 場所：広場
   - 時間：昼
   - 内容：
     - 厳造「お前は……息子だ。それだけは、変わらん」
     - 紗代「刻夜……あなたは悪くない……」
     - 小雪「兄ちゃん……約束……」

3. **処刑の瞬間**
   - 場所：広場
   - 時間：昼
   - 内容：刻夜の目の前で処刑される。刻夜の絶叫。直接的な描写は避け、刻夜の反応で悲惨さを伝える

**ポイント**：
- 厳造の「お前は息子だ」は第1章の「俺の誇りだ」と呼応
- 小雪の「約束」は果たされなかった約束の回収
- 刻夜の絶望を丁寧に描く
- 処刑の直接描写は避け、感情に焦点を当てる

### 第4話「追放」（実績：約2,300字）

**目的**：村人の裏切りと追放。「仲間」が「敵」になる絶望

**シーン構成**：
1. **太一たちの襲来**
   - 場所：村外れの納屋
   - 時間：夕方
   - 内容：処刑後、拘束されていた刻夜のもとに太一たちが来る。殴打と罵声

2. **友人たちの豹変**
   - 場所：納屋→村の中
   - 時間：夕方
   - 内容：
     - 太一「お前のせいで厳造さんが死んだんだ！」
     - 源次「お前を弟のように思っていた。だからこそ、許せない」
     - 美鈴「（目を逸らす）」——沈黙のまま去る（第1章の好意との対比）

3. **石の雨と追放**
   - 場所：村の中→村外れ
   - 時間：夕方
   - 内容：石を投げられながら村から追い出される。「二度と戻ってくるな」

4. **完全な孤立**
   - 場所：村の外→森
   - 時間：夜
   - 内容：一人で森へ向かう刻夜。師匠の姿もない

**ポイント**：
- プロローグの玄冥と同じ「石を投げられる」描写
- 昨日まで笑い合っていた人々が敵になる恐怖
- 美鈴の「目を逸らす」はプロローグの「目が合った。瞬間——逸らされた」と対比
- **リンチではなく追放**で終わる（刻夜は村人を「超越」する方向性）

---

### 第5話「襲撃」（実績：約2,500字）

**目的**：冥獣の襲撃。覚醒直前の絶望と希望の転換

**シーン構成**：
1. **彷徨い**
   - 場所：森の中
   - 時間：夜
   - 内容：追放後、一人で森を彷徨う刻夜。行く当てもなく

2. **裂け目の出現**
   - 場所：森の奥
   - 時間：深夜
   - 内容：空間に黒い亀裂が走る。冥界の瘴気が溢れ出す

3. **冥獣の襲撃**
   - 場所：森の奥
   - 時間：深夜
   - 内容：
     - 丑型（牛）の冥獣が出現
     - 刻夜は逃げられない（足を痛めている）
     - 「もう、いい。全部、終わりにしてくれ」

4. **覚醒への転換**
   - 場所：森の奥
   - 時間：深夜
   - 内容：
     - 謎の声「このまま死んで、いいのですか」
     - 刻夜「許せない……この世界の理を」
     - ギョクドウの顕現と冥獣撃破
     - 刻夜、意識を失う

**ポイント**：
- 冥獣は「世界のシステムが生んだ脅威」——刻夜が戦うべき敵の象徴
- 「許さない」の対象が村人ではなく「世界の理」に
- ギョクドウの初登場（静かに冥獣を一掃）
- 第3章（覚醒後の対話）への強い引き

**冥獣について（この話限定の設定）**：
- 外見：牛の形をした異形。黒い霧を纏い、瞳は虚ろ
- 強さ：下級冥獣（一般兵士で対処可能だが、刻夜一人では無理）
- 出現理由：「裂け目」から溢れ出した
- 詳細は `世界設定.md` の「冥獣」セクションを参照

## 文体・トーンの指針

### この章のトーン
- **第1話**：混乱、恐怖
- **第2話**：理不尽、焦り
- **第3話**：悲しみ、絶望
- **第4話**：裏切り、孤立
- **第5話**：苦痛、そして怒りへの転換

### 文体の特徴
- 一人称/三人称：三人称（刻夜視点）
- 文の長さ：短文主体。息苦しさを表現
- ダッシュ「——」の使用：衝撃、断絶を表現
- 独白の頻度：多め。刻夜の内面崩壊を描く

### 文体サンプル

**【混乱の描写】**：
```
目を開けた。

体が重い。頭が痛い。ここは——森の中だ。

なぜ、俺は森にいる。

記憶が混濁している。昨夜のことを思い出そうとして——

「……っ」

左手の甲を見た瞬間、血の気が引いた。

そこにあるべきものが——ない。
代わりに、薄く光る円形の痕跡だけが残っている。

「何だ……これは……」
```

**【処刑シーンの描写】**：
```
「父さん——！」

叫んだ。喉が裂けるほどに。

厳造はこちらを見ていた。その目には——涙はなかった。代わりに、深い慈愛があった。

「お前は……息子だ」

それが、最後の言葉だった。

刃が振り下ろされる音が聞こえた。

刻夜は目を閉じられなかった。閉じることを、許されなかった。
```

**【リンチ描写】**：
```
何度目かの蹴りが、腹に入った。

痛みはもう、感じない。感じなくなった。

「死ね」「化け物」「災厄」

声が遠い。すべてが遠い。

ああ——そうか。

俺は死ぬのか。

このまま、何もできずに。

「……もう、いい」

声が漏れた。自分の声とは思えなかった。

「全部……終わりにしてくれ……」
```

## プロローグとの対比構造

| 要素 | 玄冥（プロローグ） | 刻夜（第2章） |
|------|-------------------|-------------|
| 迫害の始まり | 「虚ろの子」と呼ばれる | 「空白」「災厄」と呼ばれる |
| 石を投げられる | 石が飛んできた | 太一たちが石を投げる |
| 信じた人の反応 | 目を逸らされた | 美鈴が目を逸らす |
| 孤立 | 独りだった | 師匠すら助けない |
| 結末 | 力の覚醒へ | 第3章で力の覚醒へ |

**読者への効果**：
- プロローグと同じ構造を認識させる
- 「また同じことが起きている」という不安
- 「刻夜は違う選択ができるのか」という期待
