# プロローグ：虚ろの子

## テーマ
「千年前、同じ絶望を味わった者がいた——」

## 概要
千年前の玄冥の悲劇を断片的に描く。刻夜と同じ「空白の命刻」を持ち、すべてを奪われた男。しかし彼は「繋がる」ではなく「統べる」を選んだ。

---

## シーン構成

### シーン1：幸せだった頃（回想）
- **場所**：どこか温かい場所（具体的には描写しない）
- **時間**：千年前・ある日
- **内容**：
  - 誰かの笑い声。温かい食事。愛する者たちの姿
  - 玄冥「（心の中で）……これが、俺の日々だった」
  - 幸福な記憶が断片的に流れる
  - **読者には「誰の視点か」を明かさない**

### シーン2：拒絶の記憶
- **場所**：村の中心
- **時間**：千年前・ある夜
- **内容**：
  - 「虚ろの子」と呼ばれ、石を投げられる
  - 「お前のせいだ」「化け物」「この世にいてはならない」
  - 愛した者たちが目を逸らす
  - 玄冥「……俺は、何も変わっていないのに」
  - **刻夜の未来と重なる描写**

### シーン3：力の覚醒と孤独
- **場所**：荒野
- **時間**：千年前・追放後
- **内容**：
  - すべてを失い、一人で立つ男
  - 「……繋がりたかった。ただ、それだけだったのに」
  - 周囲に120の気配（十大主星＋十二大従星）が顕現
  - 力を得ても、迫害は終わらなかった
  - 優しすぎた男は、復讐することもできなかった
  - **※本編では後に明かされる真実：大司星が玄冥に近づき「固定」を教えた**
  - **プロローグでは「孤独の中で自ら選んだ」ように描く（後のどんでん返しのため）**

### シーン4：世界の固定
- **場所**：天と地の狭間
- **時間**：千年前・「固定」の瞬間
- **内容**：
  - 膨大な力が世界を覆う
  - 「これで——もう誰も、俺のようには生まれない」
  - 「空白は、二度と——」
  - 十大主星・十二大従星は反対したが、絶対服従のため従うしかなかった
  - **※プロローグでは「彼らの反対」は描かない（後の伏線として温存）**
  - しかし、その瞳には涙
  - 「……すまない。俺には、これしかできなかった」

### シーン5：千年後への橋渡し
- **場所**：闇の中
- **時間**：現在（物語開始直前）
- **内容**：
  - 玄冥の声（またはナレーション）
  - 「千年が経った」
  - 「歪みは限界に達し——また一人、生まれてしまった」
  - 「空白の命刻を持つ者が」
  - 「……お前は、どうする？」
  - 「俺と同じ道を歩くか——それとも」
  - 暗転。第1章へ

## 演出上のポイント

### 意図的に隠す情報
- 玄冥の名前（「男」「彼」で通す）
- 具体的な時代背景
- 十大主星・十二大従星の名前
- 「固定」の詳細な仕組み
- **大司星の存在と裏切り**（後に明かす最大の伏線）
- **十大主星・十二大従星が反対していた事実**
- **本来の天命院が空白と協力関係にあったこと**

### 対比構造
| 玄冥（プロローグ） | 刻夜（第1章以降） |
|-------------------|------------------|
| 愛する者がいた | 養家族がいる |
| 「虚ろの子」と呼ばれた | 「空白の命刻」と呼ばれる |
| すべてを奪われた | すべてを奪われる |
| 力を得ても優しすぎて復讐できなかった | 復讐を選ぶ |
| 孤独だった（大司星に利用された） | 配下が本当に寄り添う |
| 「統べる」を選んだ（騙されて） | 「繋がる」を選べる |

### 読者への効果
1. 「この人は誰だろう？」という謎
2. 刻夜との類似性への気づき（読み進めると分かる）
3. 「刻夜も同じ道を歩むのか？」という緊張感
4. 終盤で「プロローグの男＝玄冥」と判明した時のカタルシス
5. **真実が明かされた時、玄冥への印象が「加害者」から「被害者」へ反転する**

## 文体・雰囲気

- **一人称視点**（「俺」）だが、名乗らない
- **短い文、断片的な描写**
- 詩的で抽象的なトーン
- 具体的な戦闘や術の描写は避ける
- 感情に焦点を当てる

### 参考文例

```
【シーン2のイメージ】

石が飛んできた。
頬を切り、血が流れる。

「化け物」
「災いの子」
「この世にいてはならない」

俺は——何も変わっていない。
昨日まで一緒に笑っていた。
同じ飯を食い、同じ空を見上げていた。

なのに。

「……なあ」

声をかけようとした。
あの人に。誰よりも信じていた、あの人に。

目が合った——瞬間、逸らされた。

ああ。
そうか。

俺は、独りだったんだな。
最初から、ずっと。
```

## 次章（第1章）への接続

- プロローグの最後の問いかけ「お前は、どうする？」
- 第1章冒頭：平和な日常の描写
- 読者は「この日常が壊されるのだろう」と予感しながら読む
- 緊張感を持った状態で第1章に入れる
