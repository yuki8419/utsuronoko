# 執筆ガイド：第1章「日常の終焉」

## この章の目標

1. 刻夜の「失われる前の日常」を描き、読者に感情移入させる
2. 師匠・星禍による命刻消去（封印解除）を描く
3. 「空白の命刻」という概念と、それがいかに恐れられているかを示す

**バックストーリー（執筆者用メモ）**：
- 刻夜は17年前、玄冥の血を引く両親から生まれた
- 生まれた瞬間「空白の命刻」が現れ、両親は刻夜を守るため命を落とした
- 師匠・星禍が刻夜を引き取り、封印して「普通の子」として育てた
- 刻夜はこれらの事実を知らない

## この章のプロット

### シーン1：平和な日常
- **場所**：蒼龍国辺境の村
- **時間**：夕方
- **内容**：
  - 養家族との夕食風景
  - 厳造、紗代、小雪との温かいやり取り
  - 刻夜の「ここが居場所だ」という思い

### シーン2：師匠との稽古
- **場所**：村外れの稽古場
- **時間**：翌朝
- **内容**：
  - 星禍との稽古
  - 「お前は、強くならなくていい」という謎の言葉
  - 刻夜は意味を理解できない

### シーン3：命刻消去
- **場所**：森の中
- **時間**：夜
- **内容**：
  - 星禍が刻夜を呼び出す
  - 「動くな」と言い、刻夜の手の甲に手を当てる
  - 激痛。封印が解かれ、「空白」が露わになる
  - 「これが、お前のためだと——信じたかった」
  - 刻夜は意識を失う

## 必要な設定（この章で使うもののみ）

### 命刻関連
- **命刻とは**：生まれた時に刻まれる魂の刻印。手の甲に現れる
- **空白の命刻**：何も刻まれていない状態。「災厄」として恐れられる
- この章では干支術の詳細は不要（まだ覚醒前）

### 世界設定関連
- **蒼龍国**：東の木の国。緑豊かな辺境に刻夜の村がある
- **天命院**：命刻を管理する組織（この章では名前程度でOK）

## 執筆上の注意

### 口調・文体
- **刻夜**：「俺」「です/ます」（目上に対して）。穏やかで礼儀正しい
- **厳造**：「……」が多い。短い言葉。「お前」「だな」
- **紗代**：「〜ねえ」「〜なさいな」。柔らかい
- **小雪**：「兄ちゃん」「〜だよね！」。元気
- **星禍**：「……」が多い。「〜だ」「〜しろ」。感情を抑えた口調

### 禁止事項
- 刻夜の出自（玄冥の血筋）はまだ明かさない
- 星禍の動機は明確にしない（謎のまま）
- 十大主星・十二大従星の存在は示さない

## 次章への引き

- 刻夜が意識を失った状態で終わる
- 「空白の命刻」が手の甲に刻まれている
- 翌朝、村の定期命刻検査が行われることを示唆
- 読者に「どうなってしまうのか」という緊張感を与える

## 参考：覚醒前の刻夜の描写ポイント

```
【夕食の風景】
紗代「刻夜、今日も稽古だったの？　大変ねえ」
刻夜「いえ、先生に教わるのは楽しいですから」
厳造「……ふん。お前は真面目だな。俺の若い頃とは大違いだ」
小雪「父ちゃんは今も真面目じゃないよね！」
厳造「こら、小雪！」
（笑い声が食卓に響く）
刻夜「（心の中で）……ここが、俺の居場所だ」
```

この温かさが失われることで、刻夜の絶望と怒りの深さが伝わる。
