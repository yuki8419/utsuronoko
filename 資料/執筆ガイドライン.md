# 執筆ガイドライン

読者の共感・感情移入を高めるためのポイント。

---

## 1. 刻夜の内面描写

### 基本方針
- 刻夜は「落ち着いている」キャラだが、**内心では感情が動いている**ことを示す
- 行動や台詞だけでなく、**心の中の独白**を適度に入れる

### 具体例
```
❌ 厳造が笑った。刻夜も微笑んだ。
⭕ 厳造が笑った。刻夜も微笑んだ。——こういう時間が、ずっと続けばいい。
```

### 感情表現のバランス
| シーン | 表出レベル |
|--------|-----------|
| 日常シーン | 抑制的（独白で本音を見せる） |
| 喪失シーン | 爆発的（叫び、涙、震え） |
| 復讐シーン | 冷静（怒りを内に秘める） |
| 配下との絆 | 不器用な優しさ |

---

## 2. 喪失前の「幸福感」を明示する

### 原則
**失う前に「これが大切だ」と読者に伝える**

第1章・第2章で処刑の悲しみを最大化するために：
- 刻夜が「ここにいたい」「この時間が好きだ」と明示的に思う場面を入れる
- 養家族との具体的な思い出（一緒に何をしたか）を描く
- 小雪との約束など「未来への希望」を設定する

### 第1章・第2章では達成済み
- 厳造の「お前は俺の誇りだ」
- 紗代の「ここがあんたの家」（回想）
- 小雪との森の約束

---

## 3. 敵側の「人間性」

### 原則
**完全な悪人は作らない**（大司星を除く）

### 例：太一のケース（第2章4話）
- 太一は刻夜を殴るが、それは「恐怖」から来ている
- 「俺たち、ずっとお前と一緒にいたんだぞ」→ 裏切りではなく自己防衛
- 読者は太一を憎みつつも「この世界では仕方ない」と理解できる

### 今後の敵キャラ
- 天命院の司星官たち → 「法に従っているだけ」という立場
- 四神など → それぞれの信念や過去を持たせる
- 大司星 → 唯一の「真の敵」として描く

---

## 4. 配下との関係性

### ギョクドウとの関係
- ギョクドウは「完璧な従者」だが、刻夜に対する感情は本物
- 刻夜が不器用に感謝を示す場面を入れる
- 「主従」から「絆」へ変化していく過程を描く

### 他の十大主星・十二大従星
- 全員が「刻夜のために」動く理由を個別に持たせる
- 外伝で深掘りする予定だが、本編でも軽く触れる

---

## 5. 「統べる」と「繋がる」の対比

### 刻夜の変化
| 段階 | 姿勢 |
|------|------|
| 第一部 | 復讐のために力を使う（統べる寄り） |
| 第二部 | 配下との絆に気づき始める |
| 第三部 | 「統べる」への疑問が芽生える |
| 第四部 | 玄冥との対峙で選択を迫られる |
| 第五部 | 「繋がる」を選び、人間のまま勝利 |

### 描写のコツ
- 第一部では「俺の力」「俺が統べる」という言い方
- 第五部では「俺たちの力」「俺が繋ぐ」に変化
- 変化は段階的に、読者が気づくように

---

## 6. Web小説としての工夫

### 話末の引き
- 各話の終わりに「次が読みたい」と思わせる要素を入れる
- 謎の提示、新キャラの登場、危機的状況

### テンポ
- 1話あたり2,500〜3,500字程度
- 1話1シーンを基本に
- 長い説明より、行動と会話で見せる
