# 干支組み合わせ例

## 十干（空間への干渉）

| 干 | 陰陽 | 五行 | 空間への作用 |
|----|------|------|-------------|
| 甲 | 陽 | 木 | 広げる・伸ばす |
| 乙 | 陰 | 木 | 絡める・繋ぐ |
| 丙 | 陽 | 火 | 照らす・暴く |
| 丁 | 陰 | 火 | 灯す・導く |
| 戊 | 陽 | 土 | 固める・塞ぐ |
| 己 | 陰 | 土 | 染める・浸透する |
| 庚 | 陽 | 金 | 断つ・切り裂く |
| 辛 | 陰 | 金 | 刺す・穿つ |
| 壬 | 陽 | 水 | 映す・流す |
| 癸 | 陰 | 水 | 沁みる・溶かす |

| 丑 | 陰 | 停滞・固定する |

| 卯 | 陰 | 調和・同期させる |
| 辰 | 陽 | 変容・形を変える |
| 巳 | 陰 | 蓄積・力を溜める |
| 午 | 陽 | 頂点・最大化する |
| 未 | 陰 | 熟成・安定させる |
| 申 | 陽 | 伝播・拡散する |

## 組み合わせパターン（使用例）

### 攻撃系
| 組み合わせ | 効果 | 使用場面 |
|-----------|------|---------|
| 庚×酉 | 退路が「確定的に」消える | 逃げる敵を追い詰める |
| 庚×午 | 切断力が最大化 | 強敵の防御を破る |
| 辛×寅 | 高速の刺突 | 奇襲 |

### 防御・移動系
| 組み合わせ | 効果 | 使用場面 |
|-----------|------|---------|
| 甲×子 | なかったはずの道が出現 | 脱出・奇襲 |
| 戊×丑 | 空間と時間が同時に凍結 | 絶対防御 |
| 壬×亥 | 過去の状態に戻す | 回復・修復 |

### 連携・支援系
| 組み合わせ | 効果 | 使用場面 |
|-----------|------|---------|
| 乙×卯 | 離れた味方同士が完璧に連携 | 連携攻撃 |
| 丁×巳 | 導きながら力を溜める | 大技の準備 |

### 情報・探知系
| 組み合わせ | 効果 | 使用場面 |
|-----------|------|---------|
| 丙×午 | すべてを照らし暴く | 隠れた敵の発見 |
| 癸×申 | 情報が広範囲に沁み渡る | 広域探査 |
