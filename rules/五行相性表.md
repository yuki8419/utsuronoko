# 五行相性表

## 基本属性

| 五行 | 性質 | 関連する干（陽/陰） |
|------|------|-------------------|
| 木 | 成長・拡大 | 甲（陽）/ 乙（陰） |
| 火 | 熱・光 | 丙（陽）/ 丁（陰） |
| 土 | 安定・固定 | 戊（陽）/ 己（陰） |
| 金 | 収縮・切断 | 庚（陽）/ 辛（陰） |
| 水 | 流動・浸透 | 壬（陽）/ 癸（陰） |

## 相生（力を与える）

```
木 → 火 → 土 → 金 → 水 → 木...
```

- 木生火：木は燃えて火を生む
- 火生土：火は灰となり土を生む
- 土生金：土の中から金属が生まれる
- 金生水：金属は冷えて水を生む（結露）
- 水生木：水は木を育てる

**戦闘での応用**: 相生関係の配下同士は連携しやすい

## 相剋（力を抑える）

```
木 → 土 → 水 → 火 → 金 → 木...
```

- 木剋土：木は土に根を張り養分を奪う
- 土剋水：土は水を堰き止める
- 水剋火：水は火を消す
- 火剋金：火は金属を溶かす
- 金剋木：金属（斧）は木を切る

**戦闘での応用**: 相剋関係を利用すれば弱点を突ける

## 比和（同じ属性同士）

```
木 vs 木、火 vs 火、土 vs 土、金 vs 金、水 vs 水
```

- 同じ五行同士の戦いは**純粋な力比べ**になる
- 相生・相剋のような有利不利がない
- 実力差がそのまま結果に反映される
- 技量、経験、気力の差で勝敗が決まる

**戦闘での応用**:
- 同属性の敵には、別属性の味方を当てる方が戦術的に有利
- 同属性同士で戦わせる場合は、本人の実力に自信がある時のみ

## 戦闘時チェックリスト

□ 味方の五行属性を確認
□ 敵の五行属性を「理を視る目」で把握
□ 相剋関係で弱点を突けるか？
□ 相生関係で味方を強化できるか？
□ 矛盾した描写になっていないか？
