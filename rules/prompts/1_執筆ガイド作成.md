# 執筆ガイド作成用プロンプト

```
【記入欄】ここだけ編集
部：第一部
章：5

————以下は編集不要————

【タスク】上記の章の執筆ガイドを作成してください

【手順】
1. 以下のファイルを読み込んでください：
   - settings/配下設定.md（タイムライン確認）
   - plot/parts/（上記の部）_プロット.md（この章の位置づけ）
   - plot/guides/_テンプレート.md（フォーマット）
   - 前章の執筆ガイドまたはサマリー（引き継ぎ事項）

2. 章の構成案（話数・タイトル・各話の概要）を提案してください
   ※この時点で私の承認を待ってください

3. 承認後、執筆ガイドを作成してください
   - 登場キャラの性別・外見は characters/十大主星.md 冒頭【早見表】から転記
   - 推測で書かない

4. 作成後、サブエージェントでチェックを実行してください

【ガイド作成後のチェック項目】

■ A. 設定整合性
1. キャラクター設定
   - 性別・外見が characters/十大主星.md 【早見表】と一致しているか
   - 口調サンプルが設定と一致しているか
   - 刻夜への呼称が正しいか

2. 禁止情報
   - 「玄冥」の名前が含まれていないか（第五部まで隠す）
   - 開示時期より早い情報が含まれていないか

3. タイムライン
   - settings/配下設定.md「本編タイムライン」と矛盾がないか
   - 十大主星の顕現状況が正しいか
   - 前章からの連続性があるか

■ B. 構成・論理整合性
4. プロットの整合性
   - plot/parts/（該当部）_プロット.md の方針と合っているか
   - 各話の目的が明確か
   - 話の流れに論理的な飛躍がないか

5. キャラクターの行動原理
   - 各キャラの行動に納得感があるか
   - 「このキャラがこんなことをするのは不自然」がないか

■ C. 品質・面白さ
6. 章全体の構成
   - 緊張と緩和のバランスは適切か
   - 読者を飽きさせない展開になっているか
   - 「次の章が読みたい」と思える引きがあるか

7. 各話の魅力
   - 各話に見せ場やフックがあるか
   - 冗長な話や退屈になりそうな話がないか
   - 共感できるシーンが含まれているか

8. 伏線・考察要素
   - 張る伏線が効果的か
   - 読者が「気づける余地」が残されているか

【報告形式】
■ 設定エラー：設定と矛盾する箇所 → 必ず修正
■ 論理エラー：辻褄が合わない箇所 → 必ず修正
■ 品質改善：構成・面白さの改善提案 → 検討
■ OK：問題なしの項目

【出力先】
plot/guides/（上記の章）.md
```
