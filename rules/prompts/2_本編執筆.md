# 本編執筆用プロンプト

```
【記入欄】ここだけ編集
部：第一部
章：5
話：1

————以下は編集不要————

【タスク】上記の章・話を執筆してください

【前提】
- 執筆ガイド：plot/guides/（上記の章）.md を参照
- ガイドは自己完結型なので、基本的にそのファイルだけで執筆可能

【執筆ルール】
- 文字数目安：3,000〜5,000字
- 視点：三人称限定視点（刻夜視点）
- キャラの外見・口調はガイドの「登場キャラクター」セクションに従う

【執筆後のチェック】
執筆完了後、サブエージェントで以下をチェックしてください：

■ 1. 設定整合性（つじつま）
- [ ] 顕現状態：この時点で未顕現の主星が登場していないか（settings/配下設定.md 参照）
- [ ] 能力の有無：まだ使えない能力を使っていないか
- [ ] 時系列：前話・前章との時間経過に矛盾がないか
- [ ] 場所の状態：舞台となる場所の状態が設定と一致しているか
- [ ] キャラの状態：怪我・疲労・感情状態が前話から引き継がれているか
- [ ] アイテム・装備：持っていないものを使っていないか
- [ ] 既出情報：読者が知らない情報を「知っている前提」で書いていないか

■ 2. キャラクター
- [ ] 性別・外見が characters/十大主星.md 【早見表】と一致しているか
- [ ] 口調が設定と一致しているか（語尾、一人称、敬語など）
- [ ] 性格に反する言動がないか
- [ ] キャラ同士の関係性が正しく描かれているか
- [ ] 呼称（お互いの呼び方）が設定通りか

■ 3. 禁止情報（ネタバレ防止）
- [ ] 玄冥の名前が出ていないか（第五部まで隠す）
- [ ] 星禍の真意が明かされていないか（第四部まで隠す）
- [ ] 刻夜の血筋に言及していないか（第三部〜第四部で開示）
- [ ] この章で開示予定でない伏線を先出ししていないか
- [ ] 十大主星が知らないはずの情報を知っていないか

■ 4. 共感性
- [ ] 感情の根拠：キャラの感情に「なぜそう感じるか」の理由があるか
- [ ] 変化の過程：考えや感情が変わるなら、その過程が描かれているか
- [ ] 行動の動機：行動に納得できる動機があるか
- [ ] 読者の理解：読者がキャラの立場を想像できる描写があるか
- [ ] 過剰/不足：感情表現が大げさすぎ/淡白すぎないか
- [ ] 反応の自然さ：状況に対するキャラの反応が不自然でないか
- [ ] 読者との距離：説明的になりすぎず、読者が感情移入できるか

■ 5. 物語の構成
- [ ] 前話との連続性：唐突な場面転換がないか
- [ ] この話の役割：章全体の中でこの話が果たす役割が明確か
- [ ] 引き：「次が読みたい」と思える終わり方か
- [ ] 情報量：詰め込みすぎ/スカスカになっていないか

■ 6. 文体・表現
- [ ] テンポ：冗長な箇所、飛躍している箇所がないか
- [ ] 視点ブレ：三人称限定視点（刻夜）が守られているか
- [ ] 重複表現：同じ言い回しが近くで繰り返されていないか
- [ ] 地の文と会話のバランス：どちらかに偏りすぎていないか
- [ ] 固有名詞：初出時に説明があるか、既出なら説明過多でないか

【出力先】
本編/（上記の部）/（上記の章）/（上記の話）_タイトル.md
```
