# 国家設定

## 世界の構造

この世界は**五大国**と**無数の小国・属国**で構成されている。

天命院は国家を超えた宗教的権威として全土に影響力を持つ。

---

## 五大国

### 1. 蒼龍国（そうりゅうこく）——東の大国
**五行**: 木（青龍）
**特徴**: 森林と農耕の国。最も広大な領土を持つ

| 項目 | 内容 |
|------|------|
| 位置 | 大陸東部 |
| 首都 | 翠京（すいきょう） |
| 国力 | 五大国最大。人口・兵力ともに最多 |
| 政体 | 皇帝制（天命院の影響が最も強い） |
| 特産 | 穀物、木材、絹 |
| 軍事力 | 正規軍50万。命刻兵団が主力 |

**天命院との関係**:
- 天命院の本拠地がある
- 皇帝は大司星の「推挙」がないと即位できない
- 事実上、天命院の傀儡国家

**刻夜との関係**:
- 刻夜の出身国（辺境の村）
- 最初に蹂躙される対象
- 第二部〜第三部の主戦場

---

### 2. 朱雀国（すざくこく）——南の軍事国家
**五行**: 火（朱雀）
**特徴**: 砂漠と火山の国。好戦的な軍事国家

| 項目 | 内容 |
|------|------|
| 位置 | 大陸南部 |
| 首都 | 炎都（えんと） |
| 国力 | 軍事力は五大国随一 |
| 政体 | 軍事独裁（将軍が支配） |
| 特産 | 鉱物、武器、傭兵 |
| 軍事力 | 精鋭軍30万。質で蒼龍国を凌ぐ |

**天命院との関係**:
- 形式上は従っているが、内心では反発
- 命刻による支配を嫌う武人文化
- 「強き者が正義」という価値観

**刻夜との関係**:
- 第三部〜第四部で対峙
- 一部の武将が刻夜に共鳴する可能性

---

### 3. 白虎国（びゃっここく）——西の商業国家
**五行**: 金（白虎）
**特徴**: 山岳と砂漠のオアシス都市群。商人の国

| 項目 | 内容 |

| 位置 | 大陸西部 |

| 国力 | 経済力は五大国最大 |

| 特産 | 宝石、香辛料、奴隷 |
| 軍事力 | 正規軍10万。傭兵を多用 |

**天命院との関係**:
- 金で繋がる関係
- 天命院に「寄付」することで自治を維持
- 空白の奴隷売買という闇も

**刻夜との関係**:
- 情報収集の拠点として利用
- ロクソン（魅惑）が活躍する場

### 4. 玄武国（げんぶこく）——北の守護国家
**五行**: 水（玄武）
**特徴**: 凍土と湖沼の国。防衛に特化した堅牢な国

**刻夜との関係**:
- 潜在的な同盟相手
- 第四部〜第五部で重要な役割

### 5. 黄龍国（こうりゅうこく）——中央の聖地
**五行**: 土（黄龍/麒麟）
**特徴**: 五大国の中央に位置する小国。天命院の聖地

## 天命院の支配構造

### 命刻による社会階層

```
【上位命刻】
- 戦闘向き、希少な干支
- 貴族、武将、司星官になれる
- 例：丙午（火の頂点）、庚寅（金の突破）

【中位命刻】
- 一般的な干支
- 商人、職人、農民
- 例：己未（土の癒し）、乙卯（木の敏捷）

【下位命刻】
- 弱い、または「不吉」とされる干支
- 奴隷、被差別民
- 例：癸亥（水の絶）、丁丑（火の忍耐）

【空白】
- 命刻なし
- 即処刑、または奴隷以下の扱い
```

### AI管理社会との対比

| 命刻システム | AI管理社会 |
|-------------|-----------|
| 生まれた時に命刻が決まる | 生まれた時からデータ収集 |
| 命刻で職業が制限される | AIスコアで機会が制限される |
| 変更不可能（固定） | 変更困難（履歴が残る） |
| 天命院が「正しい生き方」を決める | AIが「最適解」を提示 |
| 空白は排除される | 異常値は排除される |
| 「運命は変えられない」 | 「データは嘘をつかない」 |

## 刻夜の蹂躙計画（物語の流れ）

### 第一部：個人レベル
- 村からの追放
- 冥獣との戦い
- 覚醒と十大主星の顕現

### 第二部：地方レベル
- 蒼龍国辺境の神殿を制圧
- 下級司星官を排除
- 「魔王」の噂が広まる

### 第三部：国家レベル
- 十二司星官との対決
- 蒼龍国の地方都市を陥落
- 十二大従星の解放

### 第四部：大陸レベル
- 五大国を巻き込む戦争
- 勇者パーティとの決戦
- 天命院本部への侵攻
- **一国（蒼龍国）を事実上滅ぼす**

### 第五部：世界レベル
- 冥界への突入
- 世界の理そのものとの対決
- 新たな秩序の構築
