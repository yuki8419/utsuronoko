# 世界設定

## 命刻システムの基本

### 120干支の世界
この世界には**120の干支**が存在する。

```
【現世の60干支（命刻）】
十干（空間）× 十二支（時間）= 60通り
→ 一般人に認識できる

【冥界の60干支（冥刻）】
現世と対になる陰の干支
→ 一般人には認識できない
```

### 一般人と命刻
- **生まれた時**に60干支のうち1つが**手の甲**に刻まれる
- その命刻に応じた**力が1つ使える**
- 命刻は「魂の証」として可視化されている
- 日常的に見える場所のため、隠すには手袋などが必要

---

## 空白とは何か

### 定義
- **命刻が刻まれない存在**
- 自分では何の能力も使えない
- しかし「**世界の声が聴ける**」

### 空白の特殊能力
| 能力 | 説明 |
|------|------|
| **顕現** | 十大主星・十二大従星を呼び出せる |
| **認識** | 一般人には見えない冥刻も認識できる |
| **器** | 120干支すべてを受け入れられる唯一の存在 |

### なぜ空白だけが顕現できるのか
- 普通の人は「1つの干支」に縛られている
- 空白は「何も刻まれていない」＝**すべてを受け入れられる**
- 十大主星・十二大従星は「空白」という器にしか宿れない

## 空白の歴史

### 古代——尊ばれた時代
- 空白は「**調停者**」として尊ばれていた
- 十大主星・十二大従星を顕現し、世界のバランスを保つ
- 稀少な存在として大切にされた

### 時代を経て——恐れられる存在へ
- 空白＝120干支を従える＝**権力の一極集中**
- 王や権力者にとって脅威になる
- 「調停者」から「危険な存在」へと認識が変化
- 徐々に迫害・差別の対象に

### 玄冥の時代（千年前）
1. **幼少期の迫害**
   - 玄冥は空白として生まれた
   - まだ力に目覚める前から迫害・差別を受けた
   - 愛する者に裏切られ、すべてを失った

2. **力の覚醒**
   - 絶望の果てに十大主星・十二大従星を顕現
   - 120干支すべてを統べる力を得た

3. **玄冥の性格**
   - **優しすぎた**：力を得ても復讐・蹂躙ができなかった
   - 迫害は力を得た後も続いた
   - 「こんな苦しみを、他の空白に味わわせたくない」という自己犠牲的な思い

4. **大司星の囁き**
   - 唯一玄冥に寄り添った（ふりをした）大司星
   - 「お前のような存在が苦しまない方法がある」
   - 「世界を固定すれば、空白は二度と生まれない」と教えた

5. **玄冥の選択**
   - 大司星の言葉を信じ、世界を「**固定**」した
   - 十大主星・十二大従星は反対したが、**絶対服従**のため従うしかなかった
   - 玄冥は彼らの声を聞かなかった——聞く余裕がなかった

### 玄冥がやったこと

| 項目 | 本来の状態 | 固定後 |
|------|-----------|--------|
| **命刻の性質** | 傾向、可能性。変化しうる | 絶対的な運命。逃れられない |
| **現世と冥界** | 循環している | 分断されている |
| **空白の発生** | 自然に生まれ、調停者として機能 | 生まれないはず……だった |

### 玄冥の誤算——善意が生んだ最悪の結果

```
【玄冥の意図】
空白が生まれなければ、誰も迫害されない
            ↓
【結果】
空白は生まれ続けた（歪みの「ガス抜き」として）
            ↓
【さらに悪化】
空白は「システムエラー」として即座に殺される存在に
            ↓
【皮肉】
玄冥より酷い迫害を受ける構造を、玄冥自身が作ってしまった
```

## 現在の世界

### 空白が恐れられる理由

| 要因 | 説明 |
|------|------|
| **権力への脅威** | 120干支を従える＝誰よりも強大な力 |
| **玄冥の伝説** | 「空白が現れると世界が滅ぶ」という言い伝え |
| **天命院の教義** | 「空白は穢れ。発見次第、処刑せよ」 |
| **本能的拒絶** | 空白を見ると「何かがおかしい」と感じる |

### 空白発生の周期と崩壊

玄冥の「固定」から千年。歪みが蓄積し、空白の発生間隔が短くなっている。

| 時期 | 発生間隔 | 空白 |
|------|---------|------|
| 千年前 | — | 玄冥 |
| 800年前 | 200年 | 第2例（即処刑） |
| 600年前 | 200年 | 第3例（即処刑） |
| 400年前 | 200年 | 第4例（即処刑） |
| 200年前 | 200年 | 第5例（即処刑） |

**意味するもの**
- 世界は限界に達しつつある
- 刻夜が「今このタイミングで」覚醒した意味がある
- 対処しなければ、世界は自壊する

## 天命院

### 本来の天命院（千年前以前）
- 空白（調停者）と**協力関係**にあった
- 空白が十大主星・十二大従星を顕現し、世界のエネルギーを循環
- 天命院は命刻の管理と人々の生活を支える役割
- 大司星は空白と天命院を繋ぐ存在だった

```
【本来の世界秩序】
空白（調停者）───協力───天命院
      │                    │
      ↓                    ↓
十大主星・十二大従星    命刻の管理
      │                    │
      └──────┬──────┘
              ↓
        世界の調和
```

### 千年前の大司星——裏切り者
玄冥の時代の大司星は、玄冥を利用して世界を支配しようとした。

**表の顔**
- 迫害される玄冥に唯一寄り添った存在
- 「お前を理解できるのは私だけだ」と信頼を得た
- 「お前のような存在が苦しまない方法がある」と囁いた

**裏の顔**
- 「空白がいなければ、天命院だけで世界を支配できる」
- 玄冥の善意を利用し、「固定」の方法を教えた
- 空白の発生を防ぐことで、自分への脅威を排除しようとした

```
【大司星の企み】
玄冥の視点：
「この人だけは俺を理解してくれた」
「この人の言う通りにすれば、もう苦しむ者は生まれない」
            ↓
大司星の視点：
「空白さえいなくなれば、世界は我が手中に」
「玄冥、お前は良い駒だった」
```

### 現在の天命院
- 千年前の大司星の血筋が、現在の大司星・**萬星**
- 「空白との協力関係」という歴史は**隠蔽**されている
- 「空白は穢れ」という教義を広め、空白を排除し続けている
- 本来の「世界の調和を守る組織」から「命刻で人々を支配する組織」へと変質

### 組織構造
```
大司星（萬星）
    │
十二司星官（十二支を司る）
    │
四神＋騰蛇（五方守護）
    │
下級司星官（各地の管理）
```

### 十二大従星の封印
- 玄冥が去った後、天命院が十二大従星を封じた
- 十二司星官がそれぞれの従星を「借用」している
- 司星官を倒すと、封印が解けて従星が解放される

## 現世と冥界

### 本来の状態
- 現世（陽）と冥界（陰）は循環していた
- エネルギーが行き来し、バランスを保っていた

### 固定後の状態
- 現世と冥界が分断された
- 冥界に触れると「蝕まれる」ようになった
- 玄冥は「創星帝」として冥界に封じられた

## 十大主星・十二大従星と空白の関係

### 創世の使命
彼らには定められた使命がある：

> *「空白の王が現れたとき、我らはその手足となる」*

### なぜ空白に従うのか
- 空白だけが彼らを「顕現」させられる
- 空白は120干支すべてを受け入れられる唯一の器
- 彼らにとって空白は「主」であり「繋ぎ手」

### 絶対服従の掟
- 空白の命令には**逆らえない**
- どれほど間違っていると思っても、従うしかない
- これは「呪い」ではなく「存在の理」——空白なしには顕現できないから

### 玄冥との関係
- 玄冥も彼らを従えた
- しかし玄冥は「統べる」ことを選んだ
- **彼らは反対した**——世界を固定すれば歪みが生まれると警告した
- しかし絶対服従のため、最終的には従うしかなかった
- 玄冥は彼らの声を聞かなかった——大司星の言葉だけを信じた

### 刻夜への期待
- 千年ぶりに覚醒まで生き延びた空白
- 今度こそ「繋ぐ」道を選んでほしい
- **玄冥と同じ過ちを繰り返さないでほしい**
