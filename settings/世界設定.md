# 世界設定

## 命刻システムの基本

### 120干支の世界
この世界には**120の干支**が存在する。

```
【現世の60干支（命刻）】
十干（空間）× 十二支（時間）= 60通り
→ 一般人に認識できる

【冥界の60干支（冥刻）】
現世と対になる陰の干支
→ 一般人には認識できない
```

### 一般人と命刻
- **生まれた時**に60干支のうち1つが**手の甲**に刻まれる
- その命刻に応じた**力が1つ使える**
- 命刻は「魂の証」として可視化されている
- 日常的に見える場所のため、隠すには手袋などが必要

### 60干支の能力決定ルール

1. **十干（何を）× 十二支（どう使う）** で能力の方向性が決まる
   - 例：丙寅（ひのえとら）＝火×陽×寅 → 「炎で突破する」方向性
   - 例：癸酉（みずのととり）＝水×陰×酉 → 「本質を見切る」方向性

2. **一般人の能力**
   - 1つの干支につき1つの能力
   - 能力の強さは個人差があるが、基本的に「日常で使える程度」
   - 戦闘向きの干支もあれば、生活向きの干支もある

読者がイメージしやすいよう、代表的な干支の能力を例示する。

|------|--------|------|---------------|

| **丙寅** | 爪炎 | 爪先に小さな炎を灯す | 火起こし、暗所照明 |

**ポイント**：

   - 「理を視る目」で干支の本質を完全に把握
   - 一般人より高い精度・威力で発動可能

## 空白とは何か

### 定義
- **命刻が刻まれない存在**
- 本質は「**空っぽの器**」
- 自分自身の固有の形（命刻）を持たないため、あらゆる力を受け入れられる
- 通常の人間には耐えられない「冥界の干支（異物）」さえも宿すことができる

### 空白の特殊能力
| 能力 | 説明 |
|------|------|
| **顕現** | 十大主星・十二大従星を呼び出し、使役する |
| **認識** | 一般人には見えない冥刻（冥界の干支）も認識できる |
| **受容** | 120干支すべてを受け入れ、自らの力として振るうことができる |

### なぜ空白だけが顕現できるのか
- 普通の人は「1つの干支」という固定された形を持っている
- 形があるため、他の形（星々）を入れる隙間がない
- 空白は「無」であるため、無限の許容量を持つ
- 故に、世界のシステムからはじき出された「異物（冥界の力）」の受け皿にもなり得る

### 空白＝最強の直感的理解

**たとえ話：水と器**

```
【普通の人】
すでに「形」が決まった器（コップ、壺、皿…）
→ 自分の形に合う水しか入らない
→ 他の器の水を入れようとすると溢れる

【空白】
「形がない」器——あるいは「器そのものがない」
→ どんな形の水でも受け入れられる
→ むしろ「水そのもの」になれる
```

**もうひとつのたとえ：楽器**

```
【普通の人】
一つの楽器だけ弾ける（ピアノなら生涯ピアノ）

【空白】
どんな楽器でも弾ける
しかも「その楽器の創り手」の理解で弾く
→ 作曲家が自分で演奏するようなもの
```

**要するに**：
- 普通の人：「1つの力を、使い方を学んで使う」
- 空白：「すべての力を、本質から理解して使う」
- **制限がないから最強**、ではなく
- **制限を"理解"できるから最強**

## 空白の歴史

### 古代——尊ばれた時代
- 空白は「**調停者**」として尊ばれていた
- 十大主星・十二大従星を顕現し、世界のバランスを保つ
- 稀少な存在として大切にされた

### 時代を経て——恐れられる存在へ
- 空白＝120干支を従える＝**権力の一極集中**
- 王や権力者にとって脅威になる
- 「調停者」から「危険な存在」へと認識が変化
- 徐々に迫害・差別の対象に

### 玄冥の時代（千年前）
1. **幼少期の迫害**
   - 玄冥は空白として生まれた
   - まだ力に目覚める前から迫害・差別を受けた
   - 愛する者に裏切られ、すべてを失った

2. **力の覚醒**
   - 絶望の果てに十大主星・十二大従星を顕現
   - 120干支すべてを統べる力を得た

3. **玄冥の性格——「優しさ」と「弱さ」**
   - **心根は優しかった**：本来は争いを好まない性格
   - しかしそれは裏返せば**「決断できない弱さ」**でもあった
   - 力を得ても、迫害する者たちに立ち向かえなかった
   - 「こんな苦しみを、他の空白に味わわせたくない」——その思い自体は純粋だった
   - だが**逃避**と**自己犠牲**を混同していた

4. **大司星の囁き**
   - 唯一玄冥に寄り添った（ふりをした）大司星
   - 「お前のような存在が苦しまない方法がある」
   - 「世界を固定すれば、空白は二度と生まれない」と教えた

5. **玄冥の選択——「統べる」という逃避**
   - 大司星の言葉を信じ、世界を「**固定**」した
   - 十大主星・十二大従星は反対したが、**絶対服従**のため従うしかなかった
   - 玄冥は彼らの声を聞かなかった——聞く余裕がなかった

   **「統べる」とは何だったのか**：
   - 世界を「固定」し、自分を冥界に封じること
   - 人々と戦わず、自分から消えることで「解決」しようとした
   - これは**優しさではなく、逃避**だった
   - 戦えば勝てた。だが「傷つけること」から逃げた
   - 結果、より多くの苦しみを生んでしまった

### 玄冥がやったこと

| 項目 | 本来の状態 | 固定後 |
|------|-----------|--------|
| **命刻の性質** | 傾向、可能性。変化しうる | 絶対的な運命。逃れられない |
| **現世と冥界** | 循環している | 分断されている |
| **空白の発生** | 自然に生まれ、調停者として機能 | 生まれないはず……だった |

### 玄冥の誤算——善意が生んだ最悪の結果

```
【玄冥の意図】
空白が生まれなければ、誰も迫害されない
            ↓
【結果】
空白は生まれ続けた（歪みの「ガス抜き」として）
            ↓
【さらに悪化】
空白は「システムエラー」として即座に殺される存在に
            ↓
【皮肉】
玄冥より酷い迫害を受ける構造を、玄冥自身が作ってしまった
```

|------|------|-----------|---------|
| **下級** | 一般兵士で対処可能 | 小型、形が崩れている | 頻繁 |
| **中級** | 精鋭部隊が必要 | 十二支の特徴が明確 | 月に数回 |
| **上級** | 司星官級の力が必要 | 巨大、複数の支が混ざることも | 年に数回 |
| **災害級** | 天命の子でなければ対処不能 | 完全な十二支の姿、理性の片鱗 | 数十年に一度 |

#### 裂け目（さけめ）

冥獣が現れる場所には「裂け目」が生じる。

- **外見**: 空間に走る黒い亀裂。冥界の瘴気が漏れ出す
- **発生条件**: 歪みが蓄積した場所、大量の死が起きた場所
- **封印方法**: 天命院の術で一時的に塞げるが、根本解決にはならない
- **放置した場合**: 裂け目は広がり続け、より強力な冥獣が出現する

#### 天命院の「大義名分」

冥獣の存在は、天命院の支配を正当化する根拠となっている。

```
【天命院の主張】
「冥獣から民を守れるのは天命院だけである」
「だから民は天命院に従わねばならない」
「空白の者は冥獣を呼び寄せる穢れだ」
        ↓
【真実】
冥獣が生まれる原因は、玄冥の「固定」
その「固定」を唆したのは、千年前の大司星
つまり天命院こそが、冥獣を生み出した元凶
```

#### 勇者と冥獣

「天命の子」と勇者パーティは、冥獣討伐を通じて名声と経験を積んでいる。

- **日常任務**: 各地の裂け目を塞ぎ、冥獣を討伐
- **大任務**: 災害級冥獣との決戦
- **最終目標**: 「空白の者」の討伐——冥獣より危険な存在として

勇者にとって刻夜は「最大の冥獣」として認識されている。

#### 冥獣と冥界寄り従星の違い

冥獣と冥界寄り従星（シズメ・オサメ・サスラ）は同じ「冥界の力」を源としているが、本質が異なる。

| 項目 | 冥獣 | 冥界寄り従星 |
|------|------|-------------|
| **正体** | 冥界に蓄積した混沌が腐敗したもの | 冥界と現世の境界に存在する正当な従星 |
| **知性** | なし（本能のみ） | あり（人格と意志を持つ） |
| **発生** | 歪みの限界点で自然発生 | 世界創生時から存在 |
| **役割** | 破壊と殺戮（混沌のガス抜き） | 終末（死・墓・絶）を司る調律者 |
| **刻夜との関係** | 敵対（倒すべき脅威） | 配下（従星として仕える） |

**なぜ似て非なるのか**
- 冥獣は「固定」によって行き場を失った冥界のエネルギーが**腐敗**したもの
- 従星は本来循環の中で機能する**正当な存在**
- 冥界寄り従星は冥獣を「自分たちの**汚れた影**」と認識している
- サスラ（亥・絶）は特に冥獣を忌み嫌う——「あれは無秩序。我は無への回帰」

## 主人公の出自——玄冥の血筋

### 刻夜の血統
刻夜は**玄冥の直系の子孫**である。

```
千年前：玄冥（空白として覚醒）
    │
    └── 玄冥の妹・玄陽（げんよう）
            │
            └── 子孫が密かに生き延びる
                    │
                （千年の時が流れる）
                    │
            刻夜の両親（玄冥の血を引く末裔）
                    │
                    └── 刻夜誕生（空白として）
```

**玄陽について**
- 玄冥の妹。兄とは対照的に、普通の命刻を持って生まれた
- 兄が「虚の子」として迫害される中、唯一彼を見捨てなかった
- 玄冥が冥界へ去った後、ひっそりと生き延び、子を成した
- その血筋が千年の時を経て刻夜に繋がる

### 物語的な意味
- 刻夜は「玄冥のやり直し」
- 同じ血、同じ力、しかし**違う選択**ができる
- 「お前は玄冥と同じだ」への答え：「俺は俺だ」

## 十大主星・十二大従星と空白の関係

### 創世の使命
彼らには定められた使命がある：

> *「空白の王が現れたとき、我らはその手足となる」*

### なぜ空白に従うのか
- 空白だけが彼らを「顕現」させられる
- 空白は120干支すべてを受け入れられる唯一の器
- 彼らにとって空白は「主」であり「繋ぎ手」

### 絶対服従の掟
- 空白の命令には**逆らえない**
- どれほど間違っていると思っても、従うしかない
- これは「呪い」ではなく「存在の理」——空白なしには顕現できないから

### 玄冥との関係
- 玄冥も彼らを従えた
- しかし玄冥は「統べる」ことを選んだ
- **彼らは反対した**——世界を固定すれば歪みが生まれると警告した
- しかし絶対服従のため、最終的には従うしかなかった
- 玄冥は彼らの声を聞かなかった——大司星の言葉だけを信じた

### 刻夜への期待
- 千年ぶりに覚醒まで生き延びた空白
- 今度こそ「繋ぐ」道を選んでほしい
- **玄冥と同じ過ちを繰り返さないでほしい**
