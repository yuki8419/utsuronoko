# 生活設定

## 世界の統治構造

### 天命院の支配

天命院は世界全体を統治する宗教的・政治的な最高権力。

```
【統治構造】
天命院（中央）
    │
    ├── 大司星（頂点）
    │
    ├── 十二司星官（各地域の最高権力者）
    │
    └── 各国の王・領主（天命院に従属）
```

- 各国は独自の王や領主を持つが、天命院の権威の下にある
- 「命刻」の管理権は天命院が独占
- 逆らう国は「災厄が降りかかる」として排斥される

### 天命院と各国の関係

| 関係 | 説明 |
|------|------|
| **宗教的権威** | 「創星帝の意志を継ぐ者」として崇められる |
| **命刻の管理** | 出生届、婚姻、就職——すべてに命刻が関わる |
| **司法権** | 「空白」の処刑、命刻犯罪の裁判は天命院が担当 |
| **軍事力** | 十二司星官と下級司星官が各地に駐留 |

## 国と地域

### 世界の構成

五行に対応した五つの大国と、その周辺の小国群から構成される。

| 大国 | 五行 | 方角 | 特徴 |
|------|------|------|------|
| **蒼龍国（そうりゅうこく）** | 木 | 東 | 森と農業の国。穏やかな気質 |
| **朱炎国（しゅえんこく）** | 火 | 南 | 砂漠と火山の国。情熱的で戦闘的 |
| **黄土国（おうどこく）** | 土 | 中央 | 天命院の本拠地。最も繁栄している |
| **白銀国（はくぎんこく）** | 金 | 西 | 山岳と鉱山の国。厳格で規律重視 |
| **玄水国（げんすいこく）** | 水 | 北 | 海と湖の国。知恵と交易を重視 |

### 各国の詳細

#### 蒼龍国（木）

- **風土**: 広大な森林と肥沃な農地
- **気質**: 穏やかで協調的。成長と発展を重視
- **産業**: 農業、林業、薬草栽培
- **命刻の傾向**: 木の命刻を持つ者が多い
- **天命院との関係**: 従順。司星官の言いなり
- **物語での役割**: 刻夜の出身地。序盤の舞台

#### 朱炎国（火）

- **風土**: 火山地帯と砂漠、灼熱の大地
- **気質**: 情熱的で直情的。戦いを好む
- **産業**: 鍛冶、武器製造、傭兵派遣
- **命刻の傾向**: 火の命刻を持つ者が多い
- **天命院との関係**: 反抗的な地域もある
- **物語での役割**: 中盤の舞台。シャキとの相性◎

#### 黄土国（土）

- **風土**: 広大な平原と丘陵、肥沃な大地
- **気質**: 堅実で保守的。伝統を重視
- **産業**: 商業の中心、天命院の本部
- **命刻の傾向**: 土の命刻を持つ者が多い
- **天命院との関係**: 天命院そのものの支配地
- **物語での役割**: 終盤の舞台。天命院との最終決戦

#### 白銀国（金）

- **風土**: 険しい山岳と鉱山、冷涼な気候
- **気質**: 厳格で規律重視。裁きと秩序を重んじる
- **産業**: 鉱業、金属加工、武具製造
- **命刻の傾向**: 金の命刻を持つ者が多い
- **天命院との関係**: 協力的だが独自の規律を持つ
- **物語での役割**: 司星官との激戦地

#### 玄水国（水）

- **風土**: 海岸線と湖沼、霧の多い土地
- **気質**: 知恵を重視。柔軟で変化を恐れない
- **産業**: 漁業、交易、学問
- **命刻の傾向**: 水の命刻を持つ者が多い
- **天命院との関係**: 表向き従順、裏で独自の動き
- **物語での役割**: ギョクドウとの縁が深い地

### 刻夜の出身地

刻夜が育った村は**蒼龍国**の辺境にある。

- 天命院の目が届きにくい僻地
- 木こりや農民が暮らす小さな村
- だからこそ、師匠は刻夜を隠せた

## 干支術（かんしじゅつ）

### 概要

命刻に基づいた術——この世界における「魔法」。
60干支のエネルギーを操る技術。

### 十干と十二支の役割

干支術は「十干」と「十二支」の二つの要素で構成される。

```
【干支術の構造】

十干（空間・存在）= 術の「性質」を決める
    │
    │  「何を操るか」「どんな力か」
    │
    ├── 甲（大樹）→ 成長・拡大する木の術
    ├── 丙（太陽）→ 輝き燃え盛る火の術
    ├── 庚（鉄鋼）→ 鋭く断ち切る金の術
    └── etc.

十二支（時間・運命）= 術の「エネルギー特性・持続力」を決める
    │
    │  「どんな使い方が得意か」「どのくらい力を込められるか」
    │
    ├── 子（潜在力）→ 種子的な力。発動は遅いが成長する
    ├── 午（頂点）→ 瞬発力最大。威力は強いが持続しない
    ├── 亥（浸透力）→ じわじわ効く。持続型の術向き
    └── etc.
```

### 基本原理

```
【干支術の仕組み】
命刻（生まれ持った刻印）= 十干 × 十二支
    ↓
十干 → 使える術の「性質」が決まる（五行属性）
十二支 → 術の「得意な使い方」が決まる（エネルギー特性）
    ↓
訓練と才能で術の規模・精度が向上
```

- 全員が「自分の命刻に対応した術」を使える
- 木の命刻なら木の術、火の命刻なら火の術
- ただし、大半の人は日常生活に使う程度

### 命刻と干支術の例

| 命刻 | 十干（性質） | 十二支（エネルギー） | 干支術の特徴 |
|------|-------------|---------------------|-------------|
| 甲子 | 甲（大樹） | 子（潜在力） | じっくり育つ木の術。種から森を生やすような長期戦向き |
| 丙午 | 丙（太陽） | 午（瞬発力） | 一撃必殺の火の術。威力は最大だが持続しない |
| 壬申 | 壬（大河） | 申（鋭さ） | 切れ味鋭い水の術。一点突破型 |
| 癸酉 | 癸（雨露） | 酉（終結力） | 静かに終わらせる水の術。毒や封印向き |

### 五行別の術

| 五行 | 基本的な術 | 高度な術 |
|------|-----------|---------|
| **木** | 植物の成長促進、木材の操作 | 樹木の巨大化、森の創造 |
| **火** | 火を灯す、熱を操る | 大火球、炎の壁 |
| **土** | 土を固める、地形を変える | 地震、岩の鎧 |
| **金** | 金属の硬化、刃物の鋭利化 | 無数の刃、金属の操作 |
| **水** | 水の操作、氷の生成 | 津波、霧の操作 |

### 相生・相剋の影響

五行の相性は術にも影響する。

**相生（力を与える関係）**
- 木→火：木の術が火の術を強化
- 火→土：火の術が土の術を強化
- 土→金：土の術が金の術を強化
- 金→水：金の術が水の術を強化
- 水→木：水の術が木の術を強化

**相剋（力を抑える関係）**
- 木剋土：木の術は土の術に強い
- 土剋水：土の術は水の術に強い
- 水剋火：水の術は火の術に強い
- 火剋金：火の術は金の術に強い
- 金剋木：金の術は木の術に強い

### 十二支の影響——エネルギー特性

十二支は「術のエネルギー特性」を決定する。時間＝運命を司る十二支が、術の「使い方」を決める。

| 十二支 | 人生周期 | エネルギー特性 | 術の得意分野 | 農民の使用例 |
|--------|---------|---------------|-------------|-------------|
| 子 | 胎 | 潜在力・種子的 | 発動は遅いが成長する術 | 種まき時の発芽促進 |
| 丑 | 嬰 | 忍耐・持続型 | 長時間持続する術 | 長期保存術、土壌改良 |
| 寅 | 児 | 爆発的・瞬発力 | 一気に力を放出する術 | 害獣撃退、緊急時の火起こし |
| 卯 | 少 | 成長促進・柔軟 | 広がり浸透する術 | 作物の成長促進 |
| 辰 | 青 | 変化・膨張 | 規模を拡大する術 | 雨乞い、霧払い |
| 巳 | 壮 | 浄化・変容 | 性質を変える術 | 毒消し、水の浄化 |
| 午 | 帝 | **瞬発力最大** | 一撃必殺型。威力最大だが持続しない | 収穫時の力仕事補助 |
| 未 | 老 | 熟成・調和 | じっくり効かせる術 | 発酵促進、味の調整 |
| 申 | 病 | 伝達・拡散 | 広範囲に届く術 | 声を遠くに届ける、虫除け散布 |
| 酉 | 死 | 収束・精製 | 一点に集中する術 | 穀物の選別、金属精錬 |
| 戌 | 墓 | 守護・維持 | 状態を保つ術 | 家屋の保護結界、火の番 |
| 亥 | 絶 | 再生・浸透 | じわじわ効く術 | 畑の休眠促進、冬支度 |

**エネルギー特性の相性**:
- 対冲の十二支は「逆の特性」を持つ
  - 子（潜在力）⇔ 午（瞬発力）
  - 丑（持続）⇔ 未（熟成）
  - 寅（爆発）⇔ 申（拡散）
  - 卯（成長）⇔ 酉（収束）
  - 辰（膨張）⇔ 戌（維持）
  - 巳（変容）⇔ 亥（浸透）

**戦術的な意味**:
- 丙午（太陽×瞬発力）の命刻を持つ者は、一撃の威力が凄まじいが持久戦に弱い
- 癸亥（雨露×浸透）の命刻を持つ者は、じわじわ効く術が得意だが瞬発力に欠ける
- 甲寅（大樹×爆発力）の命刻を持つ者は、一気に木を生やして押し潰す戦法が得意

## 階級と能力差

### 能力の階級

| 階級 | 使える術 | 人口比 |
|------|---------|--------|
| **一般人** | 自分の命刻に応じた初歩的な術 | 90% |
| **術師** | 複数の術、より高度な応用 | 8% |

| **下級司星官** | 命刻の読解、複数五行の術 | 0.4% |

| **大司星** | 命刻を書き換える領域 | 1人 |

### 各階級の詳細

#### 一般人（90%）

日常生活で使う程度の術。

```
例：
- 木の命刻を持つ農民：作物の成長を少し促進
- 火の命刻を持つ料理人：竈の火を操る
- 水の命刻を持つ漁師：天候を少し読む
- 金の命刻を持つ鍛冶師：金属の硬度を調整
- 土の命刻を持つ建築家：土台を固める
```

#### 術師（8%）

訓練を積み、術を専門的に使う者。

- 戦闘員として雇われる
- 複雑な術を使える
- 各国の軍や領主に仕える

#### 下級司星官（0.4%）

天命院に仕える実務者。

- 各地の神殿に駐在
- 命刻を読み解ける
- 「空白」の発見・報告が任務

#### 十二司星官（12人）

十二支の力を「借用」している。

- 各地域の最高権力者
- 封印された十二大従星の力を使う
- 刻夜が倒すべき相手

#### 大司星（1人）

萬星のみ。命刻を書き換える力を持つ。

## 職業と暮らし

### 命刻に基づく職業選択

命刻によって「向いている職業」が決まる社会。
本来は「傾向」だったが、固定された今は「運命」。

| 五行 | 向いている職業 |
|------|---------------|
| **木** | 農民、木こり、薬師、医者 |
| **火** | 鍛冶師、料理人、兵士、傭兵 |
| **土** | 商人、建築家、役人、神官 |
| **金** | 武器職人、裁判官、衛兵、鉱夫 |
| **水** | 漁師、船乗り、学者、占い師 |

### 職業の制約

命刻による職業制限は厳しい。

- 火の命刻を持つ者が漁師になるのは「理に反する」
- 相剋の職業は「災いを招く」として禁じられる
- 違反すると天命院から処罰

### 一般人の暮らし

中世風の農村・都市生活。

**農村**
- 自給自足に近い暮らし
- 干支術で農作業を効率化
- 天命院の神殿が村の中心

**都市**
- 商業と職人が中心
- 命刻による職業区画がある
- 司星官が治安を維持

### 婚姻と命刻

結婚も命刻の相性で決まる。

- **相生の組み合わせ**: 推奨される
- **相剋の組み合わせ**: 禁止または忌避される
- 「空白」との婚姻は死罪

### 経済

- 貨幣経済が存在
- 各国で異なる通貨を使用
- 天命院が共通通貨を発行（交易用）

## 空白に対する社会の反応

### 発見時の対応

1. **発見者の義務**: 即座に神殿へ報告
2. **報告しない場合**: 一族郎党が処刑
3. **匿った場合**: 村ごと「浄化」されることも

### 「浄化」とは

空白が発見された地域を「清める」儀式。

- 実態は虐殺と焼却
- 「災厄を防ぐため」という口実
- 周辺住民も巻き込まれることがある

### 民衆の認識

| 認識 | 内容 |
|------|------|
| **恐怖** | 「空白の近くにいると災いが起きる」 |
| **嫌悪** | 「命刻のない者は人ではない」 |
| **義務感** | 「見つけたら報告するのが正しい」 |

### 刻夜の村で起きたこと

- 「空白」だと発覚した刻夜
- 村人たちは養家族を差し出した
- 「災厄を防ぐため」という正義感
- しかし実態は恐怖による集団リンチ

## 技術レベル

### 概要

中世ヨーロッパ〜中世アジア程度の技術水準。
ただし、干支術による「魔法的な代替」がある。

### 生活技術

| 分野 | 技術レベル | 干支術での代替 |
|------|-----------|---------------|
| **照明** | 蝋燭、松明 | 火の術で灯りを点ける |

干支術があるため、技術革新への意欲が薄い。

- 「術で解決できるなら機械は不要」
- 結果として科学技術は停滞
- 命刻システムが世界の発展を止めている側面も

## 社会の歪み

命刻の固定によって、社会にも歪みが蓄積している。

| 歪み | 説明 |
|------|------|
| **職業の不一致** | 命刻と本人の望みが合わない |
| **婚姻の強制** | 相性で決められた結婚への不満 |
| **才能の無駄** | 命刻に合わない才能は認められない |
| **差別の固定化** | 特定の命刻が「劣っている」とされる |
| **反乱の芽** | 命刻システムへの疑問を持つ者の増加 |

### 刻夜の戦いの意味

刻夜が目指す「理の書き換え」は、この社会構造の変革でもある。

- 命刻による差別をなくす
- 職業・婚姻の自由を取り戻す
- 「空白」も認められる世界を作る
