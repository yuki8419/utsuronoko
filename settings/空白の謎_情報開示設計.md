# 空白の謎——情報開示設計

## 概要

「空白の命刻」に関する謎は物語の核心。
**誰が何を知っているか**を厳密に管理し、段階的に真実を明かしていく。

---

## 空白に関する真実（読者への開示順）

### レベル1：表層の「常識」（第一部で開示）
**一般人が信じていること**
- 空白は「災厄」「穢れ」
- 発見次第、処刑すべき存在
- 空白がいると不幸が起こる（迷信）

### レベル2：天命院の「公式見解」（第二部で開示）
**司星官クラスが知っていること**
- 空白は千年に一度程度発生する
- 十大主星・十二大従星を従える力がある
- 「空白」が増えると世界が不安定になる
- だから排除しなければならない

### レベル3：十二司星官の「秘密」（第三部で開示）
**上位幹部が知っていること**
- 千年前の「創星帝・玄冥」も空白だった
- 玄冥が世界を「固定」した
- 十二大従星は本来自由な存在だったが、封印されている
- 自分たちは従星の力を「借りている」

### レベル4：大司星の「隠された歴史」（第四部で開示）
**萬星だけが知っていること**
- 千年前の大司星が玄冥を唆した
- 「固定」は天命院の支配を永続させるための謀略
- 空白は本来「調停者」として尊ばれていた
- 玄冥は騙された被害者

### レベル5：玄冥の「真実」（第五部で開示）
**冥界に封じられた真実**
- 空白は「世界の歪みを調整する存在」
- 固定によって歪みが蓄積し、世界は崩壊に向かっている
- 玄冥は人間をやめて「世界の楔」になった
- 解放するには120干支すべてを「繋ぐ」必要がある

---

## 十大主星が知っていること

### 確実に知っていること
- 自分たちは「十干の精神の具現」
- 空白の主に従う使命を持つ
- 千年前、玄冥という空白に仕えた
- 玄冥は「統べる」を選び、世界を固定した
- 自分たちは反対したが、止められなかった

### 曖昧にしか知らないこと
- なぜ玄冥が「統べる」を選んだのか
- 大司星の謀略の全容
- 「固定」の具体的な影響
- 玄冥の現在の状態

### 知らないこと
- 刻夜が玄冥の血筋であること（物語後半で判明）
- 「繋がる」という第三の選択肢の詳細
- 玄冥の救済方法

---

## ギョクドウの情報開示タイミング

### 第3章（覚醒直後）で伝えること
```
- あなたは「空白の命刻」を持つ者
- 私たちは十大主星。あなたの命刻に眠っていた
- あなたには十の配下がいる（私を含め）
- あなたの家族を殺したのは「この世界の理」
- 天命院がその理を維持している
```

### 第4章（絆構築）で伝えること
```
- 「理を視る目」の使い方
- 十大主星それぞれの能力の概要
- 天命院の組織構造（知っている範囲で）
- 私たちはあなたの力になりたい
```

### 第三部以降で伝えること
```
- 千年前、私たちは「玄冥」という空白に仕えた
- 玄冥は「統べる」を選び、世界を固定した
- 私たちは反対したが、止められなかった
- あなたには……違う道を選んでほしい
```

### 最後まで言わないこと
```
- 「繋がる」という選択肢の存在
  → 刻夜自身が見つけるべき答え
- 玄冥が刻夜の先祖であること
  → 物語のクライマックスで判明
```

## キャラクター別：空白への認識

### 刻夜の認識変化

| 時期 | 認識 |
|------|------|
| 第1章以前 | 空白のことを知らない（封印されていた） |
| 第2章 | 「災厄」として処刑される→理不尽だと感じる |
| 第3章 | ギョクドウから真実を聞く→怒りに変わる |
| 第二部 | 天命院こそが敵だと認識→蹂躙を開始 |
| 第三部 | 玄冥の存在を知る→自分との対比を意識 |
| 第四部 | 大司星の謀略を知る→世界の構造への怒り |
| 第五部 | 玄冥の真実を知る→「繋がる」を選択 |

### 十大主星の姿勢

**基本姿勢**：
- 刻夜に必要な情報は伝える
- ただし「答え」は与えない
- 刻夜自身が選ぶことを尊重する

**理由**：
- 玄冥の時、自分たちは反対したが聞いてもらえなかった
- 今度は「主が自分で選ぶ」ことを大切にしたい
- 誘導ではなく、支える存在でありたい

### ギョクドウの葛藤

ギョクドウは最も多くを知っているが、すべてを伝えない。

**伝えない理由**：
- 玄冥の時、情報を与えても結局「統べる」を選ばれた
- 情報より「信頼関係」が重要だと学んだ
- 刻夜には自分で答えを見つけてほしい

**内心の願い**：
```
（刻夜様……あなたは、玄冥様と同じ道を歩まないでください）
（でも、それはあなた自身が選ぶこと）
（私は……ただ、傍にいます）
```

## 情報開示の原則

### 1. 「知らない」ことを活かす
- 刻夜が知らないからこそ、読者も一緒に真実を追える
- 段階的な開示でカタルシスを生む

### 2. 「知っている」キャラの苦悩
- 十大主星は知っているが、言えない葛藤がある
- その葛藤自体がドラマになる

### 3. ミスリードを仕込む
- 天命院の「公式見解」を最初に提示
- 読者にも「空白＝悪」という印象を与える
- 後で覆すことで衝撃を与える

### 4. 刻夜の成長と連動
- 情報を得るたびに刻夜の認識が変わる
- 怒り→理解→選択という成長曲線

## 執筆時の注意

1. **ギョクドウに語らせすぎない**
   - 知っていても言わない理由を常に意識

2. **刻夜の疑問を読者の疑問に**
   - 刻夜が「なぜ？」と思うタイミングで情報を出す

3. **敵側の情報も段階的に**
   - 天命院内部の描写で、彼らの認識を示す

4. **十大主星の会話で匂わせ**
   - 「あの時と同じことを繰り返さないでほしい」など
