# 設定開示ガイド——本編での説明の仕方

## 基本方針

**設定は「説明」するな、「体験」させろ**

ラノベ読者は設定資料を読みに来ていない。物語を楽しみに来ている。
設定は「必要なときに、必要な分だけ」開示する。

---

## NG例とOK例

### ❌ NG：説明文を地の文に入れる

```
命刻とは、この世界に存在する60の干支のうちの1つが
生まれた時に手の甲に刻まれるものである。
十干と十二支の組み合わせで60通りがあり、
それぞれの命刻に応じた能力が1つ使える。
```

→ 読者は「はいはい、設定ね」と読み飛ばす

### ✅ OK：シーンの中で自然に見せる

```
「おい空白、お前の手の甲、見せてみろ」
男は腕を掴み、強引に袖をまくった。
そこには——何もなかった。
「やっぱりか。刻印がねえ」
周囲がざわめく。生まれた時に誰もが持つ命刻。
それがない者を、人々は『空白』と呼んで恐れた。
```

→ 「命刻は手の甲に刻まれる」「空白は命刻がない」が自然に伝わる

## 設定別：開示タイミングと方法

### 1. 命刻システム

**いつ開示するか**：第1章
**どう開示するか**：

| 情報 | 開示方法 |
|------|---------|
| 命刻は手の甲に刻まれる | 村人が刻夜の手を見るシーン |
| 命刻で能力が決まる | 村人が「こんな命刻じゃ大したことできねえ」と嘆く |
| 空白は命刻がない | 刻夜の手の甲に何もないことで示す |

**ポイント**：
- 60干支の一覧表を本編に入れない
- 「丙寅は火の力」程度なら会話で触れてOK
- 詳細は使うたびに少しずつ開示

---

### 2. 空白の特殊性

**いつ開示するか**：第3章（覚醒後）
**どう開示するか**：

```
「なぜ俺は……こんな力を」
『あなたは空白。すべてを受け入れる器です』
ギョクドウの声が脳裏に響く。
『普通の人は、一つの形しか持てない。
 だからこそ一つの力しか使えない。
 でもあなたは——形がない。
 だから、すべてを入れられる』
```

→ ギョクドウの説明として自然に開示

---

### 3. 天命院の存在

**いつ開示するか**：第1〜2章
**どう開示するか**：

- まず「恐ろしい権力」として登場させる
- 司星官が村に来て命刻を検査するシーン
- 村人が天命院を恐れる様子
- 詳細な組織構造は第二部以降で徐々に

**ポイント**：
- 最初から「十二司星官」「大司星」を説明しない
- 読者には「なんかヤバい組織」という印象だけ与える

---

### 4. 十大主星・十二大従星

**いつ開示するか**：登場時
**どう開示するか**：

```
光の中から、女が現れた。
白銀の髪、月のように冷たい美貌。
『我が名はギョクドウ。癸の化身にして、智を司る者』
刻夜を見下ろす瞳は、どこか悲しげだった。
『千年ぶりですね——我が主よ』
```

→ 「十大主星とは〜」という説明ではなく、ギョクドウ個人として登場

**ポイント**：
- 十大主星の一覧を本編に入れない
- 登場するたびに「○○を司る者」と自己紹介
- 読者は自然と「十人いるのか」と理解する

---

### 5. 玄冥の過去

**いつ開示するか**：第三部〜第五部
**どう開示するか**：

- プロローグで断片的に匂わせる（名前は伏せる）
- 第三部でギョクドウが「千年前の主」について語る
- 第四部で大司星の謀略が判明
- 第五部で玄冥本人と対面し、全容が明かされる

**ポイント**：
- **絶対に第一部で全部説明しない**
- 読者の「知りたい」を引っ張る
- 玄冥の選択の理由は、刻夜が追体験する形で理解させる

---

## 設定開示の鉄則

### 1. 「読者が今知る必要があるか？」を問う

- 今の場面で必要ない設定は書かない
- 「後で重要になる」は開示の理由にならない
- 後で重要になるなら、後で開示すればいい

### 2. キャラの口から語らせる

- 地の文で説明するより、会話で開示
- 説明役キャラを作らない（ギョクドウを説明botにしない）
- 「教える側」と「教わる側」の関係性も描く

### 3. 「知らない」ことも武器

- 刻夜が知らないから、読者も一緒に驚ける
- 「後から判明する」はカタルシスになる
- 最初から全部説明すると、驚きがなくなる

### 4. 能力は「使うとき」に説明

```
❌ 「丙寅の命刻を持つ者は、炎を操る力を持つ」（事前説明）

✅ 男が叫び、拳を振り上げた。
   その瞬間、炎が腕を覆う。丙寅——火を纏う力。
   だが。
  「遅い」
   刻夜の声と同時に、男の身体が宙を舞った。
```

## 各部での設定開示量の目安

| 部 | 新設定 | 開示方法 |
|----|--------|---------|
| 第一部 | 命刻、空白、天命院の存在 | シーンの中で自然に |
| 第二部 | 天命院の組織構造、司星官 | 敵として戦いながら |
| 第三部 | 従星の解放、玄冥の存在 | ギョクドウの回想 |
| 第四部 | 大司星の謀略、刻夜の血統 | 敵との対話で判明 |
| 第五部 | 玄冥の真実、世界の真相 | 玄冥本人との対面 |

## 設定を説明したくなったときのチェックリスト

1. [ ] この情報は「今」必要か？
2. [ ] 地の文ではなくキャラの言葉で伝えられないか？
3. [ ] シーンの中で「見せる」ことはできないか？
4. [ ] 読者が「なぜ？」と思うタイミングか？
5. [ ] 説明を削っても物語は成立するか？

5つ中3つ以上チェックが付かなければ、その説明は**今は要らない**。

---

## 覚えておくこと

> **設定の9割は、読者に見せなくていい**

世界設定.mdに書いてあることは「作者が把握しておくべきこと」であり、
「読者に説明すべきこと」ではない。

物語の中で自然に見せられる1割だけを、
必要なタイミングで、必要な分だけ開示する。

それが「設定が重い作品」を「読みやすい作品」にする唯一の方法。
