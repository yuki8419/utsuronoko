# 戦闘システム

本作における戦闘の基本ルールと、五行相生相剋を活用した戦術。

---

## 基本原則

### 干支術（かんしじゅつ）
命刻に基づいた術——この世界における「魔法」。60干支のエネルギーを操る技術。

### 十干と十二支——戦闘における二つの軸

戦闘は「空間（十干）」と「時間（十二支）」の二軸で決まる。

```
【戦闘の二軸】

┌─────────────────────────────────────┐
│  十干（空間・存在）= 術の「性質」    │
│  「何で攻撃するか」「どんな力か」    │
│                                     │
│  → 五行相生相剋で有利不利が決まる   │
└─────────────────────────────────────┘
              ×
┌─────────────────────────────────────┐
│  十二支（時間・運命）= 術の「運用」  │
│  「どう使うか」「どんな戦い方か」    │
│                                     │
│  → エネルギー特性で戦術が決まる     │
└─────────────────────────────────────┘
```

### 基本原理
```
【干支術の仕組み】
命刻（十干 × 十二支）
    ↓
十干 → 術の「性質」（五行属性）
十二支 → 術の「運用」（エネルギー特性）
    ↓
訓練と才能で術の規模・精度が向上
```

- 全員が「自分の命刻に対応した術」を使える
- 木の命刻なら木の術、火の命刻なら火の術
- ただし、大半の人は日常生活に使う程度

### 戦闘における十二支の重要性——五行本能

十二支は五行に属し、それぞれの五行が持つ「本能」が戦闘スタイルを決定する。

#### 五行本能と十二支の対応

| 五行 | 本能 | 十二支 | 戦闘スタイル |
|------|------|--------|-------------|
| 木 | **守備本能** | 寅・卯 | 防御、持久、盾役。仲間を守り、耐え抜く |
| 火 | **伝達本能** | 巳・午 | 瞬発、速攻、突進。情報と炎を広げる |
| 土 | **引力本能** | 丑・辰・未・戌 | 拘束、重厚、中央。敵を引き寄せ固定する |
| 金 | **攻撃本能** | 申・酉 | 一撃必殺、暗殺、破壊。精密かつ冷酷 |
| 水 | **習得本能** | 亥・子 | 浸透、適応、消耗戦。じわじわと削る |

| 十二支 | 五行 | 本能 | 戦闘スタイル |
|--------|------|------|-------------|
| 子 | 水陽 | 習得 | 潜在力・始動。種子のように力を溜め、一気に芽吹かせる |

#### 五行本能の戦術的意味

| 本能 | 得意 | 苦手 | 対策 |
|------|------|------|------|
| 守備（木） | 持久戦、防御 | 速攻、奇襲 | 金属性で切り崩す |
| 伝達（火） | 速攻、突破 | 持久戦、消耗戦 | 水属性で消火する |
| 引力（土） | 拘束、重厚 | 機動戦、浸透 | 木属性で根を張る |
| 攻撃（金） | 一撃必殺、精密 | 乱戦、持久 | 火属性で溶かす |
| 習得（水） | 適応、消耗戦 | 短期決戦 | 土属性で堰き止める |

**戦術例**:
- 丙午の敵（火・伝達）と戦うなら、水属性で消火し持久戦に持ち込む
- 癸亥の敵（水・習得）と戦うなら、土属性で堰き止め短期決戦を挑む
- 庚寅の敵（木・守備）と戦うなら、金属性で防御を切り崩す

## 五行相生相剋

戦闘における最も重要な要素。五行の相性が勝敗を大きく左右する。

### 相生（そうしょう）——力を与える関係

```
木 → 火 → 土 → 金 → 水 → 木
```

| 相生 | 理由 | 戦闘での効果 |
|------|------|-------------|
| 木生火 | 木は燃えて火を生む | 木の術で火の術を強化 |
| 火生土 | 火は灰となり土を生む | 火の術で土の術を強化 |
| 土生金 | 土の中から金属が生まれる | 土の術で金の術を強化 |
| 金生水 | 金属は冷えて水を生む | 金の術で水の術を強化 |
| 水生木 | 水は木を育てる | 水の術で木の術を強化 |

**戦術的応用**:
- 連携攻撃で相生を利用すると威力が増す
- 例：ギョクドウ（水）がカンサク（木）を強化

### 相剋（そうこく）——力を抑える関係

```
木 → 土 → 水 → 火 → 金 → 木
```

| 相剋 | 理由 | 戦闘での効果 |
|------|------|-------------|
| 木剋土 | 木は土に根を張り養分を奪う | 木の術は土の術に強い |
| 土剋水 | 土は水を堰き止める | 土の術は水の術に強い |
| 水剋火 | 水は火を消す | 水の術は火の術に強い |
| 火剋金 | 火は金属を溶かす | 火の術は金の術に強い |
| 金剋木 | 金属（斧）は木を切る | 金の術は木の術に強い |

**戦術的応用**:
- 相手の五行に対して有利な属性で攻撃する
- 例：火の司星官にはギョクドウ（水）を当てる

### 比和（ひわ）——力が重なる関係

同じ五行同士の戦い（木 vs 木、火 vs 火など）。

| 比和 | 戦闘での効果 |
|------|-------------|
| **対等** | 相性による有利不利がない。純粋な「力量（レベル）」と「戦術」の勝負になる |
| **共鳴** | 互いの術が干渉し合い、威力が跳ね上がったり、暴走したりするリスクがある |
| **吸収** | 格上の術者が、格下の同属性の術を取り込んでしまうことがある |

**戦術的応用**:
- 相手と同じ属性をぶつけることで、相剋による不利（弱点）を消す
- 自分の術を味方の同属性の術に重ねて、規模を拡大する（共鳴）

## 刻夜の戦闘能力

### 「理を視る目」

刻夜の固有能力。発動時、瞳が金色に変わる。

| 能力 | 効果 |
|------|------|
| 本質看破 | 相手の命刻、五行、弱点が見える |
| 時機把握 | 最適な攻撃タイミングが分かる |
| 因果認識 | 攻撃の軌道、結果が予測できる |
| 相性分析 | 誰を当てれば有利か瞬時に判断 |

### 采配の重要性

刻夜自身の戦闘力は高くないが、「理を視る目」による采配が最大の武器。

```
【采配の例】
刻夜「ギョクドウ、左翼を霧で覆え」
刻夜「シャキ、右から突撃——三秒後に敵は防御を解く」
刻夜「カンサク、正面で受けろ。五秒耐えれば勝てる」
```

| 段階 | 戦力 |
|------|------|

## 冥界の力の代償

### 冥界の干支
- 現世の60干支と対になる陰の干支
- 本来は対等だったが、分断後は「毒」のように作用する

### 使用時の代償

| 使用頻度 | 代償 |
|---------|------|
| 初回 | 軽い疲労 |
| 数回 | 髪に銀が混じり始める |
| 頻繁 | 瞳が冷たくなっていく |
| 多用 | 体が蝕まれる |
| 限界 | 人間性を失う危機 |

### 刻夜への影響
- 冥界寄りの十二大従星（シズメ、オサメ、サスラ）を使うと代償が発生
- 四神を倒すたびに境界が薄くなり、代償が増す
- 最終的には「人間をやめる」か「繋ぐ」かの選択を迫られる

## 戦闘シーンの構成

### 基本構成

1. **状況把握**: 刻夜が「理を視る目」で分析
2. **采配**: 最適な配置と戦術を指示
3. **展開**: 五行相生相剋を活かした攻防
4. **転機**: 敵の奥の手、または味方の連携
5. **決着**: 相剋の有利、または采配の勝利

### 十二司星官戦の構成例

```
【導入】
司星官の五行と封じている従星を確認
刻夜が「理を視る目」で弱点を分析

【第一幕】
相剋を利用した攻撃
しかし、司星官は「借用している十二支の力」で対抗

【第二幕】
借用の力に苦戦
```
