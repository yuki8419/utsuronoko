# 配下設定

## 概要

刻夜の配下は「十大主星」と「十二大従星」を頂点とした階層構造を持つ。
各主星・従星は固有の眷属を持ち、**任意のタイミングで呼び出すことができる**。

---

## 階層構造

```
刻夜（主）
 │
 ├─ 十大主星（10人）─── 最高幹部、人格あり
 │    └─ 各主星の眷属 ─── 主星ごとに異なる配下
 │
 └─ 十二大従星（12人）── 幹部、封印中→解放で加入
      └─ 各従星の眷属 ─── 従星ごとに異なる配下（解放後）
```

---

## 顕現の条件【SSOT】

十大主星が刻夜の前に現れるには、**2つの条件**が必要。

### 1. 必要条件：力の蓄積

| 状態 | 顕現可能数 |
|------|-----------|
| 覚醒直後（第2章） | 1人（ギョクドウのみ） |
| 外伝期（1年かけて回復） | 徐々に増加 |
| 第4章開始時 | 全10人 |

**力の増加要因**：
- 時間経過（刻夜の命刻が安定する）
- 決断を重ねる（意思の力が増す）
- 戦闘経験（力を使うことで器が広がる）

### 2. トリガー：本質との共鳴

力が十分でも、**刻夜の心がその主星の「本質」と共鳴しなければ顕現しない**。

| 主星 | 本質 | 共鳴する刻夜の心 |
|------|------|----------------|
| ギョクドウ | 智・慈愛 | 「生きたい」という本能（覚醒時に自動共鳴） |
| シロク | 献身・生活 | 「ここで生きる」——生活を始める意思 |
| カンサク | 信義・守護 | 「守りたい」——失いたくない者ができた |
| チョウジョ | 孤独・救済 | 孤独を受け入れる——世界に拒絶された自分を認める |
| シャキ | 闘志・破壊 | 「戦いたい」——攻撃への衝動 |
| ケンギュウ | 忠節・正義 | 「正しくなくてもやる」——偽善でない決意 |
| リュウコウ | 好奇・混沌 | 「面白そう」——前向きな興味 |
| セキモン | 調和・繋がり | 「仲間」と呼ぶ——他者を認める |
| ホウカク | 自由・軽さ | 「笑えよ」——空気を変えたい |
| ロクソン | 魅惑・外交 | 「この世界で生きていく」——外部との繋がりを求める |

### なぜギョクドウだけが最初に顕現したか

- ギョクドウの本質は「慈愛」——主を守りたい、幸せにしたい
- 刻夜が死にかけた瞬間、「生きたい」という本能が発動
- これが最も根源的な欲求であり、ギョクドウと自動的に共鳴した
- 他の主星の本質は「生きたい」より高次の欲求（生活、守護、闘志など）

### 執筆時の注意

- 顕現シーンでは「なぜこの主星なのか」を刻夜の心理変化で示すこと
- 力の蓄積は明示しなくてよい（時間経過で自然に増える）
- 共鳴は「決断」「感情の変化」「言葉」で表現する

## 眷属の共通設定

### 出自
- 各主星・従星の「力の一部」が形を取った存在
- 刻夜の命刻に眠っていた（主星・従星と同じ）

### 顕現ルール
- **十大主星・十二大従星は、自分の眷属を任意のタイミングで呼び出せる**
- 上限数は決まっている（例：シャキなら刃兵60体まで）
- 必要な時に必要な数だけ呼び出すことが可能
- 全員を常時展開する必要はない

### 維持コスト
- 眷属を顕現させている間、主星・従星の力を消費する
- 長時間の大量展開は主星・従星を疲弊させる
- 刻夜の力が増すと、維持コストが軽減される
- 戦闘など負荷が高い状況では消耗が激しい

### 再顕現
- 倒された眷属は、時間経過で再び呼び出せるようになる
- 再顕現には主星・従星の力を消耗する
- 大量に倒されると、しばらく戦力が低下する

### 特徴
- 主星・従星への絶対服従
- 刻夜の命令は主星・従星を通じて伝わる（直接命令も可）
- 感情・知性は眷属の種類によって異なる

### 呼び方
- 刻夜：眷属を直接呼ぶことは少ない。主星に指示を出す
- 十大主星：自分の眷属を「我が〇〇」と呼ぶことも

### 演出例
```
シャキ「（獰猛な笑み）……出ろ、刃兵」
黒い霧から次々と双剣の兵士が現れる。
シャキ「全員だ。六十、残らず」

ギョクドウ「影衆、三だけ。偵察に」
三つの影が音もなく闇に溶けていく。
```

## 備考

- 本編では「すでに揃っている」前提で描写
- 個々の眷属にスポットを当てることは少ない（背景・群衆として描写）
- 外伝で顕現の過程を丁寧に描く
- 十二大従星の眷属は、解放イベント時に別途設定
